**C. Palindromic Times**

【个人定数】PST 6【链接】https://www.luogu.com.cn/problem/CF108A

给一个时间，问下一个形成`AB:BA`形式的时间是什么

*解：*直接转成string处理

```c++
// 处理输入 注意char转int有偏移
// 附ASCII: 48:0, 65:A, 97:a
string inp;
cin >> inp;
int h = (inp[0]-48)*10+(inp[1]-48);
int m = (inp[3]-48)*10+(inp[4]-48);
```
```c++
// 判断部分 手动补0 借助string下标判断位数 我觉得比题解拆int方便点
string ms = (m<10 ? "0" : "") + to_string(m);
string mh = (h<10 ? "0" : "") + to_string(h);
if (ms[0]==mh[1] && ms[1]==mh[0]) {
    cout << mh << ":" << ms << endl;
    return;
}
```

**E. Domino Effect**

【个人定数】PRS 8【链接】https://www.luogu.com.cn/problem/CF405B

多米诺骨牌模拟，题目问没倒下的骨牌数量，没受力或受力平衡的骨牌不会倒下。

*解：* 答案为`out`，先判断最左或最右侧没受力的牌`out+=所有没受力的牌`，然后，如果RL区间的大小为偶数，说明中间有个元素与RL的距离相同，然后`out++`。

**F. Fox and Cross**

【个人定数】PRS 8【链接】https://www.luogu.com.cn/problem/CF389B

判断一个大的点阵图形是否由一个十字`+`拼成而没有重叠、没有其他结构，是的话`Yes`不然`No`，这题我想的很复杂。最后看题解了。

*解：* 删除所有的十字图形，假如最终还有图案，说明`No`

```c++
// 判断十字的代码
bool condition = v[i][j]==s && v[i-1][j]==s && v[i+1][j]==s && v[i][j-1]==s && v[i][j+1]==s;
```

最终判断是否还有图案，只要在输入的时候计数，然后删除的时候`-5`即可

**H. Dima and To-do List**

【个人定数】PRS 8+【链接】https://www.luogu.com.cn/problem/CF366B

题目意思感觉很费解。我直接贴一个题解，[链接](https://www.luogu.com.cn/problem/solution/CF366B)

*解：* 先分组，`n=6`，`k=2`的时候分成`1__1__`、`_1__1_`、`__1__1`，输出三种情况中的最小总责备力度即可

>设我们从第 pos 个任务开干。
>
>那么，责备发生的时刻就是 pos,pos+k,pos+2k,pos+3k,pos+4k，每次加 k。所以我们只需要记录每一组的则被力度和，再找出力度最小的答案即可。

**I. Levko and Permutation**

【个人定数】PRS 8+【链接】https://www.luogu.com.cn/problem/CF361B

给`n`、`v`，有由整数1~n构成的数组`l`，重新排列数组，使`gcd(元素, 元素下标+1)>1`的项为`v`，输出排列后的数组`l`。没有就输出`-1`

*解：* `元素=1` 或 `元素下标+1=1` 会始终使 `gcd=1`。让数组`l`变为这样即可：`n-k, 下标, 下标, ..., 下标, 下标+1, 下标+1, ..., 下标+1`，因为从第二项起，`gcd(n, n+1)=1`且`gcd(n, n)=n≠1`，这样来控制数量，而且数字不会重复，不会超出[1, n]范围。
