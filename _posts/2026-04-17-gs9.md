## 9-1：多元函数基本概念

### 第一大题：填空题 (ﾉ´ヮ`)ﾉ*: ･ﾟ

**(1) 复合函数代入法**

**题目：** 设 $f(x, y) = 2x + y^2$，则 $f[xy, f(x,y)] =$ ______

**怎么做：**

1. 原函数是 $f(x, y) = 2x + y^2$，意思是：**2乘以第一个位置的东西，加上第二个位置的东西的平方**。
    
2. 现在第一个位置放的是 $xy$，第二个位置放的是 $f(x,y)$。
    
3. 我们把它们塞进去：$2(xy) + [f(x,y)]^2$
    
4. 把 $f(x,y)$ 原本的式子代入进去：$2xy + (2x+y^2)^2$。

**(2) 求定义域**

**题目：** $z = \frac{\sqrt{x-y^2}}{\ln(1-x^2-y^2)}$ 的定义域是 ______

**怎么做：**

找定义域就是要让这个式子“有意义”，要闯过三道关卡！(ง •_•)ง

1. **根号关：** 根号下的东西不能是负数，所以 $x - y^2 \ge 0$，推导出 $x \ge y^2$。
    
2. **对数关：** $\ln$ 括号里的东西必须大于 0，所以 $1 - x^2 - y^2 > 0$，推导出 $x^2 + y^2 < 1$。
    
3. **分母关：** 分母不能等于 0！即 $\ln(1-x^2-y^2) \neq 0$。因为 $\ln(1) = 0$，所以括号里的 $1 - x^2 - y^2 \neq 1$，也就是 $x^2 + y^2 \neq 0$。这就意味着 $x$ 和 $y$ 不能同时为 0。
    
    把这三个条件合在一起，就是你写的：$\{(x,y) \mid x \ge y^2 \text{ 且 } 0 < x^2+y^2 < 1\}$。
    
    _这题逻辑非常严密，你写的完全对！(๑˃ᴗ˂)ﻭ_
    

**(3) 求二重极限**

**题目：** $\lim_{(x,y)\to(0,0)} \frac{\sin(xe^y)}{x} =$ ______

**怎么做：**

用到**等价无穷小**的魔法啦！✨

1. 当 $x \to 0$ 且 $y \to 0$ 时，$e^y \to e^0 = 1$。
    
2. 所以 $xe^y \to 0 \times 1 = 0$。
    
3. 我们知道当小星星 $\to 0$ 时，$\sin(\text{小星星}) \sim \text{小星星}$。所以 $\sin(xe^y) \sim xe^y$。
    
4. 原式就变成了 $\lim_{(x,y)\to(0,0)} \frac{xe^y}{x}$。把 $x$ 约掉，就是 $\lim_{(x,y)\to(0,0)} e^y = e^0 = 1$。
    
    _这题你也做对啦！旁边还写了等价无穷小的提示，超棒！(≧◡≦)_
    

**(4) 找间断点集合**

**题目：** $z = \frac{1}{x^2+xy}$ 的间断点的集合是 ______

**怎么做：**

分式的间断点就是让**分母等于0**的点！💥

1. 令分母 $x^2 + xy = 0$。
    
2. 提取公因式：$x(x+y) = 0$。
    
3. 所以要么 $x = 0$，要么 $x+y = 0$（也就是 $x = -y$）。
    
    写成集合就是：$\{(x,y) \mid x = 0 \text{ 或 } x = -y\}$。
    
    _完美答对！(✧ω✧)_

### 第二大题：换元法求函数表达式 (๑˃̵ᴗ˂̵)و

**题目：** 设 $f(x+y, \frac{y}{x}) = x^2 - y^2$，试求 $f(x,y)$。

**怎么做：**

这种题需要用到**换元法**！

1. **设替身：** 令 $a = x+y$， $b = \frac{y}{x}$。
    
2. **解出真正的 x 和 y：**
    
    - 由 $b = \frac{y}{x}$ 得到 $y = bx$。
        
    - 把 $y = bx$ 代入 $a = x+y$，得到 $a = x + bx = x(1+b)$。
        
    - 所以 $x = \frac{a}{1+b}$。
        
    - 那么 $y = bx = b \cdot \frac{a}{1+b} = \frac{ab}{1+b}$。
        
3. **代回原式拼凑：** * 我们要求 $x^2 - y^2$。你可以用平方差公式 $(x+y)(x-y)$ 会算得更轻松哦！
    
    - $(x+y)$ 就是 $a$。
        
    - $(x-y) = \frac{a}{1+b} - \frac{ab}{1+b} = \frac{a-ab}{1+b} = a \cdot \frac{1-b}{1+b}$。
        
    - 相乘得到：$a \cdot a \cdot \frac{1-b}{1+b} = a^2 \frac{1-b}{1+b}$。
        
4. **换回 x 和 y：** 刚才算出来 $f(a,b) = a^2 \frac{1-b}{1+b}$，现在把 $a, b$ 重新写成 $x, y$（这只是个符号替换）：
    
    最终结果 $f(x,y) = x^2 \frac{1-y}{1+y}$。
    
    _你的草稿写得非常详细，最后的 $x^2$ 写得稍微有点小，不过完全是正确答案！太强了！(☆▽☆)_

### 第三大题：代入法求未知函数 ٩(◕‿◕)۶

**题目：** 设 $z = \sqrt{y} + f(\sqrt{x}-1)$，当 $y=1$ 时，$z=x$，求 $z$ 和 $f(x)$。

**怎么做：**

这题像破案一样，要先根据线索找出 $f(x)$ 的真面目！🔍

1. **用线索找 $f(x)$：**
    
    - 题目说当 $y=1$ 时，$z=x$。我们把 $y=1$ 和 $z=x$ 代入原式。
        
    - $x = \sqrt{1} + f(\sqrt{x}-1)$，也就是 $x = 1 + f(\sqrt{x}-1)$。
        
    - 移项得到：$f(\sqrt{x}-1) = x - 1$。
        
2. **换元法揭开 $f(x)$：**
    
    - 令括号里的 $t = \sqrt{x}-1$，那么 $\sqrt{x} = t+1$，两边平方得到 $x = (t+1)^2$。
        
    - 代入上面式子：$f(t) = (t+1)^2 - 1 = t^2 + 2t + 1 - 1 = t^2 + 2t$。
        
    - 把 $t$ 换成 $x$，得到 $f(x) = x^2 + 2x$。（第一问完成！）🎉
        
3. **求完整的 z：**
    
    - 现在我们知道 $f(\text{某个东西}) = (\text{某个东西})^2 + 2 \times (\text{某个东西})$ 了。
        
    - 原式 $z = \sqrt{y} + f(\sqrt{x}-1)$
        
    - 把 $\sqrt{x}-1$ 塞进刚才找到的法则里：
        
        $z = \sqrt{y} + (\sqrt{x}-1)^2 + 2(\sqrt{x}-1)$
        
    - 展开：$z = \sqrt{y} + (x - 2\sqrt{x} + 1) + 2\sqrt{x} - 2$
        
    - 抵消一下：$-2\sqrt{x}$ 和 $+2\sqrt{x}$ 没啦！$1 - 2 = -1$。
        
    - 最后得到：$z = \sqrt{y} + x - 1$。
        
        _这道大题你全对！卷面推导过程无懈可击！(o´∀`o)_

**(3) 夹逼定理（放缩法）**

**完整题目：** 计算极限 $\lim_{(x,y)\to(+\infty, +\infty)} \left( \frac{xy}{x^2 + y^2} \right)^{xy}$

**解题思路 & 步骤：**

这题你的直觉超级准！用到了基本不等式 $x^2 + y^2 \ge 2xy$，然后用“夹逼定理”就能秒杀它。

1. **找范围：** 因为 $x, y \to +\infty$，所以 $x > 0, y > 0$。
    
    根据 $x^2 + y^2 \ge 2xy$，可以得到：$0 < \frac{xy}{x^2 + y^2} \le \frac{1}{2}$。
    
2. **整体放缩：** 因为题目是一个指数函数，并且指数 $xy > 0$，所以我们可以把不等式同时加上指数 $xy$：
    
    $$0 < \left( \frac{xy}{x^2 + y^2} \right)^{xy} \le \left( \frac{1}{2} \right)^{xy}$$
    
3. **求极限：** 当 $(x,y)\to(+\infty, +\infty)$ 时，$xy \to +\infty$。
    
    我们看最右边：$\lim_{(x,y)\to(+\infty, +\infty)} \left( \frac{1}{2} \right)^{xy} = 0$。
    
4. **得出结论：** 根据夹逼定理，被夹在 $0$ 和 $0$ 中间的式子，极限只能是 $0$。
    
    所以原式 $= 0$。

### 第五大题：证明极限不存在 (ง •_•)ง

**知识点提示：** 证明二重极限不存在的最常用绝招就是**“换两条路走”**！如果沿着路径A走得到的极限，和沿着路径B走得到的极限不一样，那就说明整体极限不存在啦！

**(1) 证明题第一道**

**完整题目：** 5. 试证明下列极限不存在：(1) $\lim_{(x,y)\to(0,0)} \frac{\sin(x - y)}{x + y}$

**解题思路 & 步骤：**

（你的解法完美无瑕，直接照抄你的思路！）

1. **路径一：沿着 $x = 0$ 趋近于 $(0,0)$**
    
    把 $x = 0$ 代入原式，式子变成只关于 $y$ 的极限：
    
    $$\lim_{y\to 0} \frac{\sin(0 - y)}{0 + y} = \lim_{y\to 0} \frac{\sin(-y)}{y}$$
    
    因为 $\sin(-y) = -\sin(y)$，且当 $\star \to 0$ 时 $\sin(\star) \sim \star$，所以极限为 $-1$。
    
2. **路径二：沿着 $y = 0$ 趋近于 $(0,0)$**
    
    把 $y = 0$ 代入原式，式子变成只关于 $x$ 的极限：
    
    $$\lim_{x\to 0} \frac{\sin(x - 0)}{x + 0} = \lim_{x\to 0} \frac{\sin(x)}{x}$$
    
    根据重要极限，这个结果为 $1$。
    
3. **得出结论：**
    
    因为 $-1 \neq 1$，沿着不同路径趋近的极限值不相等，所以原极限不存在。
    
    _满分证明过程！(_^▽^_)_
    

**(2) 证明题第二道**

**完整题目：** 5. 试证明下列极限不存在：(2) $\lim_{(x,y)\to(0,0)} \frac{x^4 + y^2}{x^2 + y^4}$

**解题思路 & 步骤：**

（依然是沿着两条简单的坐标轴走！）

1. **路径一：沿着 $x = 0$ 趋近于 $(0,0)$**
    
    把 $x = 0$ 代入原式：
    
    $$\lim_{y\to 0} \frac{0 + y^2}{0 + y^4} = \lim_{y\to 0} \frac{1}{y^2}$$
    
    当 $y \to 0$ 时，分母趋于 0，所以整个式子趋于 $\infty$。
    
2. **路径二：沿着 $y = 0$ 趋近于 $(0,0)$**
    
    把 $y = 0$ 代入原式：
    
    $$\lim_{x\to 0} \frac{x^4 + 0}{x^2 + 0} = \lim_{x\to 0} x^2$$
    
    当 $x \to 0$ 时，极限为 $0$。
    
3. **得出结论：**
    
    一个是 $\infty$，一个是 $0$，结果不相同，所以原极限不存在。

#### 3. “先代后导”的终极考验 🔥(ง'̀-'́)ง

**完整题目：** 设 $f(x,y) = e^{xy}\sin\pi y + (x-1)\arctan\sqrt{\frac{x}{y}}$，求 $f_x(1,1)$ 及 $f_y(1,1)$。

**【解析】** 你的思路是对的！先代入再求导能省一半的力气！但是卷面上有一点小小的计算失误哦，我们来抓虫子！🐛

**求 $f_x(1,1)$：**

1. 先令 $y=1$，代入得：$f(x,1) = e^x\sin\pi + (x-1)\arctan\sqrt{x}$。
    
2. 因为 $\sin\pi = 0$，所以第一项没了！$f(x,1) = (x-1)\arctan\sqrt{x}$。
    
3. 对 $x$ 求导（乘法求导法则）：
    
    $f_x(x,1) = 1 \cdot \arctan\sqrt{x} + (x-1) \cdot (\arctan\sqrt{x})'$。
    
4. **高能预警⚡**：我们要求的是 $x=1$ 时的值，此时后面的 $(x-1)$ 刚好等于 $0$！所以后面那一串根本不用算！
    
    $f_x(1,1) = \arctan\sqrt{1} + 0 = \arctan 1$。
    
5. **纠错点**：你卷子上划掉的地方好像写了 $0$，但其实 **$\arctan 1 = \frac{\pi}{4}$** 哦！（$\tan 45^\circ = 1$ 嘛）😄
    

**求 $f_y(1,1)$：**

1. 先令 $x=1$，代入得：$f(1,y) = e^y\sin\pi y + 0 \cdot \arctan(\dots) = e^y\sin\pi y$。
    
2. 对 $y$ 求导（乘法法则）：
    
    $f_y(1,y) = e^y\sin\pi y + e^y \cdot (\pi\cos\pi y)$。
    
3. 代入 $y=1$：
    
    $f_y(1,1) = e^1\sin\pi + e^1 \cdot \pi\cos\pi = 0 + e \cdot \pi \cdot (-1) = \mathbf{-\pi e}$。
    
4. **纠错点**：$\cos\pi = -1$，所以最后结果有一个负号哦！你卷子最底下写了 $\pi \cdot e$，差一个负号就完美啦！( ；∀；)

**9. 题目:** 设 $u = y g(\frac{x}{y})$， $g$ 具有二阶连续导数，求 $\frac{\partial^2 u}{\partial x \partial y}$。

- **求 $u_x$:** 视 $y$ 为常数。设 $\star = \frac{x}{y}$。
    
    $u_x = y \cdot g'(\star) \cdot \star_x = y \cdot g'(\star) \cdot \frac{1}{y} = g'(\star)$。
    
- **求 $u_{xy}$:** 对 $g'(\star)$ 求 $y$ 的偏导。
    
    $u_{xy} = \frac{\partial}{\partial y}[g'(\star)] = g''(\star) \cdot \star_y = g''(\frac{x}{y}) \cdot (-\frac{x}{y^2})$。
    
- **结论:** 整理后得出 $-\frac{x}{y^2} g''(\frac{x}{y})$。(-‿◦)

**10. 题目:** $z = f[\varphi(x-y), \sin y]$，其中 $f$ 具有二阶连续偏导数， $\varphi$ 具有二阶连续导数，求 $\frac{\partial^2 z}{\partial x \partial y}$。

- **求 $z_x$:** 设 $\star = \varphi(x-y), \triangle = \sin y$。视 $y$ 为常数。
    
    $z_x = f_1 \cdot \star_x + f_2 \cdot \triangle_x = f_1 \cdot \varphi'(x-y) \cdot 1 + 0 = f_1 \cdot \varphi'(x-y)$。
    
- **求 $z_{xy}$:** 对上式求 $y$ 的偏导，需用**乘法法则**。
    
    $z_{xy} = \frac{\partial}{\partial y}(f_1) \cdot \varphi'(x-y) + f_1 \cdot \frac{\partial}{\partial y}[\varphi'(x-y)]$。
    
- **分别计算:**
    
    第一块 $\frac{\partial}{\partial y}(f_1)$：链式法则。
    
    $f_{11} \cdot \star_y + f_{12} \cdot \triangle_y = f_{11} \cdot \varphi'(x-y) \cdot (-1) + f_{12} \cdot \cos y$。
    
    第二块 $\frac{\partial}{\partial y}[\varphi'(x-y)]$：直接求导。
    
    $\varphi''(x-y) \cdot (-1) = -\varphi''(x-y)$。
    
- **合并结论:** 把两块代回去。
    
    $z_{xy} = [-f_{11} \varphi'(x-y) + f_{12} \cos y] \cdot \varphi'(x-y) + f_1 \cdot [-\varphi''(x-y)]$

## 9-5：隐函数的求导公式

**1. 求下列隐函数的导数、偏导数或全微分：**

**(1) 题目：** 设 $\frac{x}{y} + \ln(zy) = 0$，求 $\frac{\partial z}{\partial x}$，$\frac{\partial z}{\partial y}$；

**【解析】**

设 $F = \frac{x}{y} + \ln z + \ln y = 0$

$F_x = \frac{1}{y}$， $F_y = -\frac{x}{y^2} + \frac{1}{y}$， $F_z = \frac{1}{z}$

套公式：

$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{z}{y}$

$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = \frac{z(x-y)}{y^2}$ (≧◡≦)

**(2) 题目：** 设函数 $z = z(x,y)$ 由方程 $z = x + y\varphi(z)$ 确定，其中 $\varphi$ 有连续导数，求 $dz$。

**【解析】**

设 $F = x + y\varphi(z) - z = 0$

$F_x = 1$， $F_y = \varphi(z)$， $F_z = y\varphi'(z) - 1$

套公式：

$z_x = -\frac{F_x}{F_z} = \frac{1}{1 - y\varphi'(z)}$

$z_y = -\frac{F_y}{F_z} = \frac{\varphi(z)}{1 - y\varphi'(z)}$

全微分 $dz = z_x dx + z_y dy = \frac{dx + \varphi(z)dy}{1 - y\varphi'(z)}$ (=ﾟωﾟ)ﾉ

**2. 求下列隐函数的二阶导数或偏导数：**

**(1) 题目：** 设 $\ln\sqrt{x^2+y^2} = \arctan\frac{y}{x}$，求 $\frac{d^2y}{dx^2}$；

**【解析】**

设 $F = \frac{1}{2}\ln(x^2+y^2) - \arctan\frac{y}{x} = 0$

分别求导并通分：$F_x = \frac{x+y}{x^2+y^2}$， $F_y = \frac{y-x}{x^2+y^2}$

一阶导 $\frac{dy}{dx} = -\frac{F_x}{F_y} = \frac{x+y}{x-y}$

求二阶导（使用除法法则）：

$\frac{d^2y}{dx^2} = \left(\frac{x+y}{x-y}\right)' = \frac{(1+y')(x-y) - (x+y)(1-y')}{(x-y)^2} = \frac{2xy' - 2y}{(x-y)^2}$

把前面的 $y'$ 代进去化简：

$\frac{d^2y}{dx^2} = \frac{2(x^2+y^2)}{(x-y)^3}$ d(-_^)

**(2) 题目：** 设 $e^z - xyz = 0$，求 $\frac{\partial^2 z}{\partial x \partial y}$。

**【解析】**

设 $F = e^z - xyz = 0$。由题意已知 $e^z = xyz$。

$F_x = -yz$， $F_y = -xz$， $F_z = e^z - xy = xyz - xy = xy(z-1)$

$z_x = -\frac{F_x}{F_z} = \frac{z}{x(z-1)}$

$z_y = -\frac{F_y}{F_z} = \frac{z}{y(z-1)}$

算混合偏导，把 $z_y$ 对 $x$ 求导（视 $y$ 为常数，注意 $z$ 是含 $x$ 的函数）：

$\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial}{\partial x}\left[\frac{1}{y} \frac{z}{z-1}\right] = \frac{1}{y} \frac{z_x(z-1) - z \cdot z_x}{(z-1)^2} = -\frac{z_x}{y(z-1)^2}$

把 $z_x$ 代入：

$\frac{\partial^2 z}{\partial x \partial y} = -\frac{z}{xy(z-1)^3}$ (☆ω☆)

**3. 求下列方程组所确定的函数的导数或偏导数：**

**(1) 题目：** $\begin{cases} z = x^2 + y^2 \\ x^2 + 2y^2 + z^2 = 1 \end{cases}$，求 $\frac{dz}{dx}$，$\frac{dy}{dx}$；

**【解析】**

两边直接对 $x$ 求导：

① $z' = 2x + 2yy'$

② $2x + 4yy' + 2zz' = 0 \Rightarrow x + 2yy' + zz' = 0$

把 ① 变形成 $2yy' = z' - 2x$，丢进 ② 里：

$x + (z' - 2x) + zz' = 0 \Rightarrow z'(1+z) = x \Rightarrow \frac{dz}{dx} = \frac{x}{1+z}$

代回求 $y'$：

$2yy' = \frac{x}{1+z} - 2x = \frac{-x(1+2z)}{1+z} \Rightarrow \frac{dy}{dx} = -\frac{x(1+2z)}{2y(1+z)}$ (_^▽^_)

**(2) 题目：** $\begin{cases} xu - yv = 0 \\ yu + xv = 1 \end{cases}$，求 $\frac{\partial u}{\partial x}$，$\frac{\partial v}{\partial y}$。

**【解析】**

求 $u_x$：两个方程对 $x$ 求导（视 $y$ 为常数）：

① $u + xu_x - yv_x = 0$

② $yu_x + v + xv_x = 0$

①乘 $x$，②乘 $y$，相加消掉 $v_x$：

$(x^2+y^2)u_x + xu + yv = 0 \Rightarrow \frac{\partial u}{\partial x} = -\frac{xu+yv}{x^2+y^2}$

求 $v_y$：两个方程对 $y$ 求导（视 $x$ 为常数）：

③ $xu_y - v - yv_y = 0$

④ $u + yu_y + xv_y = 0$

③乘 $y$，④乘 $x$，相减消掉 $u_y$：

$-(x^2+y^2)v_y - vy - ux = 0 \Rightarrow \frac{\partial v}{\partial y} = -\frac{ux+vy}{x^2+y^2}$ (o^▽^o)

**4. 题目：** 设函数 $z = z(x,y)$ 是由方程 $F(\frac{y}{x}, \frac{z}{x}) = 0$ 所确定，其中 $F$ 具有连续偏导数，且 $F'_2 \neq 0$，证明：$x \frac{\partial z}{\partial x} + y \frac{\partial z}{\partial y} = z$。

**【解析】**

设 $\star = \frac{y}{x}, \triangle = \frac{z}{x}$。令 $F_1, F_2$ 为对外层第一、第二位置的偏导。

方程两边对 $x$ 求导：

$F_1 \cdot \left(-\frac{y}{x^2}\right) + F_2 \cdot \frac{xz_x - z}{x^2} = 0 \Rightarrow xz_x - z = y\frac{F_1}{F_2} \Rightarrow xz_x = z + y\frac{F_1}{F_2}$

方程两边对 $y$ 求导：

$F_1 \cdot \frac{1}{x} + F_2 \cdot \frac{z_y}{x} = 0 \Rightarrow z_y = -\frac{F_1}{F_2}$

代入要证明的等式左边：

$x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = \left(z + y\frac{F_1}{F_2}\right) + y\left(-\frac{F_1}{F_2}\right) = z$。

左右相等，证明完毕！(｀・ω・´)ゞ

## 9-6：多元函数微分学的几何应用

**1. 题目：** 求曲线 $\begin{cases} x = a \sin^2 t \\ y = b \sin t \cos t \\ z = c \cos^2 t \end{cases}$ 在 $t = \frac{\pi}{4}$ 处的切线方程及法平面方程.

- **找切点：** 代入 $t = \frac{\pi}{4}$，得坐标 $(\frac{a}{2}, \frac{b}{2}, \frac{c}{2})$。
    
- **找切向量 $\vec{v}$：** 对 $t$ 求导。
    
    $x'(t) = a \sin 2t \Rightarrow x'(\frac{\pi}{4}) = a$

    $y'(t) = b \cos 2t \Rightarrow y'(\frac{\pi}{4}) = 0$

    $z'(t) = -c \sin 2t \Rightarrow z'(\frac{\pi}{4}) = -c$

    得 $\vec{v} = (a, 0, -c)$。
    
- **切线方程：** $\frac{x - \frac{a}{2}}{a} = \frac{y - \frac{b}{2}}{0} = \frac{z - \frac{c}{2}}{-c}$ （分母为 0 表示该坐标无变化）。
    
- **法平面方程：** $a(x - \frac{a}{2}) + 0(y - \frac{b}{2}) - c(z - \frac{c}{2}) = 0 \Rightarrow ax - cz + \frac{c^2 - a^2}{2} = 0$。(✧∀✧)
    

**2. 题目：** 求曲线 $\begin{cases} x^2 + z^2 = 10 \\ y^2 + z^2 = 10 \end{cases}$ 在点 $(1, 1, 3)$ 处的切线方程及法平面方程.

- **找切向量 $\vec{v}$：** 两曲面交线，切向量等于两曲面法向量的**叉乘**。
    
    设 $F = x^2 + z^2 - 10$, $G = y^2 + z^2 - 10$。
    
    梯度 $\nabla F = (2x, 0, 2z)$，$\nabla G = (0, 2y, 2z)$。
    
    代入点 $(1, 1, 3)$ 得 $\nabla F = (2, 0, 6)$，$\nabla G = (0, 2, 6)$。
    
    $\vec{v} = \nabla F \times \nabla G = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ 2 & 0 & 6 \\ 0 & 2 & 6 \end{vmatrix} = (-12, -12, 4)$。化简取平行向量 $\vec{v} = (3, 3, -1)$。
    
- **切线方程：** $\frac{x - 1}{3} = \frac{y - 1}{3} = \frac{z - 3}{-1}$。
    
- **法平面方程：** $3(x - 1) + 3(y - 1) - 1(z - 3) = 0 \Rightarrow 3x + 3y - z - 3 = 0$。(=ﾟωﾟ)ﾉ
    

**3. 题目：** 求椭球面 $x^2 + 2y^2 + z^2 = 1$ 上平行于平面 $x - y + 2z = 0$ 的切平面方程.

- **列方程组：** 设切点 $(x_0, y_0, z_0)$。曲面法向量 $\vec{n} = (2x_0, 4y_0, 2z_0)$。已知平面法向量 $\vec{n}_0 = (1, -1, 2)$。
    
    两平面平行 $\Rightarrow \vec{n} \parallel \vec{n}_0$。得比例式：
    
    $\frac{2x_0}{1} = \frac{4y_0}{-1} = \frac{2z_0}{2} = k$
    
    解得 $x_0 = \frac{k}{2}$, $y_0 = -\frac{k}{4}$, $z_0 = k$。
    
- **求常数 $k$：** 代入椭球面方程。
    
    $(\frac{k}{2})^2 + 2(-\frac{k}{4})^2 + k^2 = 1 \Rightarrow \frac{11}{8}k^2 = 1 \Rightarrow k = \pm \sqrt{\frac{8}{11}} = \pm \frac{2\sqrt{22}}{11}$。
    
- **组装方程：** 所求平面与原平面平行，设为 $x - y + 2z = D$。
    
    把切点代入求 $D$：$D = x_0 - y_0 + 2z_0 = \frac{11}{4}k = \pm \frac{\sqrt{22}}{2}$。
    
- **最终方程：** $x - y + 2z \pm \frac{\sqrt{22}}{2} = 0$。(_^▽^_)
    

**4. 题目：** 证明: 曲面 $\sqrt{x} + \sqrt{y} + \sqrt{z} = \sqrt{a} \ (a > 0)$ 上任一点处的切平面在各坐标轴上的截距之和等于 $a$.

- **设切点：** 任取点 $P(x_0, y_0, z_0)$，满足 $\sqrt{x_0} + \sqrt{y_0} + \sqrt{z_0} = \sqrt{a}$。
    
- **求切平面：** 曲面法向量 $\vec{n} = (\frac{1}{2\sqrt{x_0}}, \frac{1}{2\sqrt{y_0}}, \frac{1}{2\sqrt{z_0}})$。
    
    切平面方程：$\frac{1}{2\sqrt{x_0}}(x - x_0) + \frac{1}{2\sqrt{y_0}}(y - y_0) + \frac{1}{2\sqrt{z_0}}(z - z_0) = 0$
    
    化简得：$\frac{x}{\sqrt{x_0}} + \frac{y}{\sqrt{y_0}} + \frac{z}{\sqrt{z_0}} = \sqrt{x_0} + \sqrt{y_0} + \sqrt{z_0} = \sqrt{a}$。
    
- **求截距：**
    
    令 $y=0, z=0$，得 $X$ 轴截距 $x_{int} = \sqrt{a}\sqrt{x_0}$。
    
    同理，$Y$ 轴截距 $y_{int} = \sqrt{a}\sqrt{y_0}$，$Z$ 轴截距 $z_{int} = \sqrt{a}\sqrt{z_0}$。
    
- **求和：** $x_{int} + y_{int} + z_{int} = \sqrt{a}(\sqrt{x_0} + \sqrt{y_0} + \sqrt{z_0}) = \sqrt{a} \cdot \sqrt{a} = a$。证明完毕！(｀_´)ゞ
