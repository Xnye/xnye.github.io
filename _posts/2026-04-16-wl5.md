# 第五章：刚体转动

***本文大部由Ai生成*** *仅作个人备份*

---

**1. 翻译法：直线运动变转动**
做题时，把牛顿力学的公式全“翻译”过来就能用：
*   质量 $m$ $\to$ 转动惯量 $J$ (惯性大小)
*   力 $F$ $\to$ 力矩 $M = F \times \text{力臂}$
*   速度 $v$ $\to$ 角速度 $\omega$ (关系式 $v = \omega r$)
*   加速度 $a$ $\to$ 角加速度 $\alpha$ (关系式 $a = \alpha r$)
*   牛顿第二定律 $F=ma$ $\to$ $M = J\alpha$

**2. 必背转动惯量 $J$**
不用现推，直接背这三个最常用的：
*   圆盘/圆柱绕中心轴：$J = \frac{1}{2}mR^2$
*   细杆绕一端：$J = \frac{1}{3}mL^2$
*   质点/子弹/小球绕固定轴：$J = mr^2$

**3. 最强杀手锏：角动量守恒**
*   **触发条件**：题目里出现“子弹打木块/杆”、“人走动”、“黏在一起”、“非弹性碰撞”且**有固定转轴**。
*   **公式**：初始角动量总和 = 最终角动量总和 ($L_1 = L_2$)。
*   **怎么算 $L$**：
    *   刚体：$L = J\omega$
    *   直线飞行的质点(子弹)：$L = mv \times \text{力臂}$

**4. 积分套路：处理变速转动**
*   **触发条件**：题目给的阻力矩 $M$ 跟角速度 $\omega$ 有关（比如 $M = -k\omega$ 或 $-k\omega^2$），求时间。
*   **标准动作**：写出牛二定律 $J\frac{d\omega}{dt} = M$，把 $\omega$ 和 $dt$ 分离到两边，两边同时积分。

# 🎯 例题

#### 1. [选择题] 坐标向量求速度
**题目**：刚体以 60rev/min 绕 z 轴转动 ($\vec{\omega}$ 沿 z 正向)，P点位置 $\vec{r} = 3\vec{i} + 4\vec{j} + 5\vec{k}$。求此时 P 点速度。
(A) $\vec{v} = 94.2\vec{i} + 125.6\vec{j} + 157.0\vec{k}$
(B) $\vec{v} = -25.1\vec{i} + 18.8\vec{j}$
(C) $\vec{v} = -25.1\vec{i} - 18.8\vec{j}$
(D) $\vec{v} = 31.4\vec{k}$
**图像说明**：无
**解题步骤**：
1. 换算角速度单位：$60 \text{ rev/min} = 60 \times \frac{2\pi}{60} \text{ rad/s} = 2\pi \text{ rad/s}$。
2. 写成向量：$\vec{\omega} = 2\pi\vec{k}$。
3. 代入叉乘公式：$\vec{v} = \vec{\omega} \times \vec{r}$。
4. $(2\pi\vec{k}) \times (3\vec{i} + 4\vec{j} + 5\vec{k}) = 6\pi(\vec{k}\times\vec{i}) + 8\pi(\vec{k}\times\vec{j}) + 0 = 6\pi\vec{j} - 8\pi\vec{i}$。
5. 代入 $\pi \approx 3.14$ 计算：$-8\pi \approx -25.1$， $6\pi \approx 18.8$。所以 $\vec{v} = -25.1\vec{i} + 18.8\vec{j}$。**选 (B)**。

#### 2. [选择题] 子弹穿杆
**题目**：静止细棒 ($M, L, J=\frac{1}{3}ML^2$) 绕 O 点转动。子弹 ($m, v$) 射穿自由端，速度变 $\frac{1}{2}v$，求棒的角速度。
选项：(A) $\frac{mv}{ML}$ (B) $\frac{3mv}{2ML}$ (C) $\frac{5mv}{3ML}$ (D) $\frac{7mv}{4ML}$
**图像说明**：俯视图，一根杆绕左端转，子弹从右端穿过。
**解题步骤**：
1. 典型的碰撞题 $\to$ **角动量守恒**。
2. 碰撞前：只有子弹有角动量 $L_0 = mvL$。
3. 碰撞后：子弹残余角动量 $L_1 = m(\frac{1}{2}v)L$，杆获得角动量 $L_{rod} = J\omega = (\frac{1}{3}ML^2)\omega$。
4. 列等式：$mvL = \frac{1}{2}mvL + \frac{1}{3}ML^2\omega$。
5. 解得：$\frac{1}{2}mvL = \frac{1}{3}ML^2\omega \implies \omega = \frac{3mv}{2ML}$。**选 (B)**。

#### 3. [选择题] 碰撞中的守恒律
**题目**：小球水平打击悬挂的细杆（非弹性碰撞），在此过程中系统哪个量守恒？
选项：(A)只有机械能守恒 (B)只有动量守恒 (C)只有对转轴O的角动量守恒 (D)都守恒
**图像说明**：杆悬挂于O点，球从左边飞来撞击。
**解题步骤**：
1. “非弹性碰撞” $\to$ 机械能必定损失。
2. 转轴 O 处会产生强大的约束反力（外力） $\to$ 系统合外力不为0，动量不守恒。
3. 转轴 O 处的反力穿过轴心，力臂为0，不产生外力矩 $\to$ 合外力矩为0，**角动量守恒**。**选 (C)**。

#### 4. [选择题] 两子弹打圆盘
**题目**：圆盘绕水平固定轴转动 ($\omega_0$)，两颗质量、速度相同，方向相反并在一直线上的子弹射入并留在盘内。盘的角速度如何变化？
选项：(A)增大 (B)不变 (C)减小 (D)不能确定
**图像说明**：圆盘转动，两颗子弹从左右两边对称地打向圆盘。
**解题步骤**：
1. 两颗子弹在同一直线上且方向相反，它们对轴心的总角动量为0（互相抵消）。
2. 打入后，系统总角动量不变（因为外加角动量为0）。
3. 但子弹留在了盘里，系统的总转动惯量 $J$ 变大了 ($J_{new} = J_{disk} + 2mr^2$)。
4. 根据 $L = J\omega$，因为 $L$ 不变、$J$ 增大，所以 $\omega$ 必定减小。**选 (C)**。

#### 5. [选择题] 滑轮角加速度比较
**题目**：A、B相同的定滑轮，A挂质量为 M 的物体，B受拉力 F=Mg。比较角加速度 $\alpha_A, \alpha_B$。
选项：(A)-(D) 比较大小
**图像说明**：A滑轮挂实体物块M，B滑轮直接被力F往下拉。
**解题步骤**：
1. B滑轮：绳子拉力直接是外力 $Mg$，提供力矩 $M_B = MgR$。所以 $J\alpha_B = MgR$。
2. A滑轮：物块往下加速掉落（失重状态），所以绳子的拉力 $T < Mg$。提供给滑轮的力矩 $M_A = TR < MgR$。
3. 因为作用在A上的力矩更小，所以 $\alpha_A < \alpha_B$。**选 (C)**。

#### 6. [选择题] 细杆持续转动的临界条件
**题目**：杆(长$l$, 质量$m$)绕离一端 $l/4$ 处的轴转动。给起始角速度 $\omega_0$，要使其持续转动不往复摆动，求 $\omega_0$。
选项：(A)-(D)
**图像说明**：一根杆，转轴在上端往下四分之一处。
**解题步骤**：
1. “持续转动”意味着它至少能转到最高点。这是转动中的机械能守恒题。
2. 找质心位置：质心在杆中心（离端点 $l/2$），转轴在 $l/4$，所以质心到转轴距离 $d = l/4$。
3. 算转动惯量（平行轴定理）：$J = J_c + md^2 = \frac{1}{12}ml^2 + m(\frac{l}{4})^2 = \frac{7}{48}ml^2$。
4. 从最低点到最高点，质心升高了 $2d = l/2$。重力势能增加 $mg(l/2)$。
5. 动能全部转化为势能：$\frac{1}{2}J\omega_0^2 \ge mg\frac{l}{2}$。
6. 代入：$\frac{1}{2}(\frac{7}{48}ml^2)\omega_0^2 \ge \frac{1}{2}mgl \implies \omega_0 \ge \sqrt{\frac{48g}{7l}} = 4\sqrt{\frac{3g}{7l}}$。**选 (A)**。

#### 7. [填空题] 梯子平衡问题
**题目**：均匀梯子($l, W$)靠光滑墙和地面，下端连弹簧($k$)，与地面成 $\theta$ 角平衡。求(1)地面作用力 (2)墙作用力 (3)物理量关系。
**图像说明**：梯子斜靠，底部有弹簧拉着。
**解题步骤**：
1. (1) 竖直方向受力平衡：地面的支持力刚好等于重力，所以地面的作用力为 **$W$**。
2. (2) 水平方向平衡：墙的推力 $N$ 等于弹簧的拉力。竖直时为原长，说明弹簧被拉伸了底部的横向距离 $x = l\cos\theta$（注意看图，角 $\theta$ 是和地面的夹角，底部移动距离是 $l\cos\theta$。等等，题目图里没标角，但文字说“与地面成 $\theta$ 角”，竖直放置时底部靠墙，现在移开了 $l\cos\theta$）。所以墙的作用力等于弹簧力，为 **$kl\cos\theta$**。
3. (3) 对触地点列力矩平衡：顺时针(重力) = 逆时针(墙推力)。重力力臂为 $\frac{l}{2}\cos\theta$，墙推力力臂为 $l\sin\theta$。所以 $W\frac{l}{2}\cos\theta = (kl\cos\theta) \times l\sin\theta$。化简得关系式：**$W = 2kl\sin\theta$**。

#### 8. [填空题] 联合加速系统
**题目**：滑块A($m_A$)在桌面，重物B($m_B$)悬挂，滑轮C($m_C, J=\frac{1}{2}m_CR^2$)。无摩擦。求A的加速度 $a$。
**图像说明**：桌面物块A连绳子过滑轮拉物块B。
**解题步骤**：
1. 把直线运动全部转换成等效质量：滑轮的转动惯量等效为挂在绳子上的质量 $\frac{J}{R^2} = \frac{1}{2}m_C$。
2. 把 A、B、C 看作一个整体，提供动力的只有 B 的重力 $m_Bg$。
3. 整体的总等效质量 = $m_A + m_B + \frac{1}{2}m_C$。
4. 根据整体牛顿第二定律 $F = M_{总}a$：$m_Bg = (m_A + m_B + \frac{1}{2}m_C)a$。
5. 结果：**$a = \frac{m_Bg}{m_A + m_B + m_C/2}$**。

#### 9. [填空题] 子弹射杆
**题目**：匀质杆($l, m', J=\frac{1}{3}m'l^2$)竖直下垂。子弹($m, v_0$)打在 $OA = 2l/3$ 处嵌入。求角速度。
**图像说明**：杆悬挂，子弹水平击中中下部。
**解题步骤**：
1. **角动量守恒套路**。
2. 初始角动量（只有子弹）：$L = mv_0 \times (力臂) = m v_0 (\frac{2}{3}l)$。
3. 最终转动惯量（系统）：$J_{总} = J_{杆} + J_{子弹} = \frac{1}{3}m'l^2 + m(\frac{2}{3}l)^2 = (\frac{1}{3}m' + \frac{4}{9}m)l^2$。
4. 令 $L = J_{总}\omega$，求得：$\omega = \frac{\frac{2}{3}mv_0l}{(\frac{1}{3}m' + \frac{4}{9}m)l^2} = $ **$\frac{6mv_0}{(3m' + 4m)l}$**。

#### 10. [填空题] 变力矩刹车
**题目**：飞轮转动惯量 $J$，初态 $\omega_0$。阻力矩 $M \propto \omega^2$，设 $M = -k\omega^2$。当 $\omega = \frac{1}{3}\omega_0$ 时，求角加速度 $\beta$ 和时间 $t$。
**解题步骤**：
1. 第一空（求 $\beta$）：直接代入 $M = J\beta$。所以 $-k\omega^2 = J\beta$。把 $\omega = \frac{1}{3}\omega_0$ 代入得大小：**$\frac{k\omega_0^2}{9J}$** (若问带有方向的代数值则加负号)。
2. 第二空（积分求 $t$）：
   $$J\frac{d\omega}{dt} = -k\omega^2 \implies \frac{d\omega}{\omega^2} = -\frac{k}{J}dt$$
   两边积分：$\int_{\omega_0}^{\omega_0/3} \omega^{-2} d\omega = \int_0^t -\frac{k}{J} dt$
   左边积分为 $[-\frac{1}{\omega}]_{\omega_0}^{\omega_0/3} = -\frac{3}{\omega_0} - (-\frac{1}{\omega_0}) = -\frac{2}{\omega_0}$。
   等式两边消去负号：$\frac{2}{\omega_0} = \frac{k}{J}t \implies t = $ **$\frac{2J}{k\omega_0}$**。

#### 11. [填空题] 杆的摩擦力矩
**题目**：杆($m, l$)在桌面绕一端转动，摩擦系数 $\mu$。求摩擦力矩。
**解题步骤**：
1. 杆上每一处摩擦力力臂不同，需微积分。
2. 距转轴 $x$ 处取微元 $dx$，质量 $dm = \frac{m}{l}dx$。
3. 微元受到的摩擦力 $df = \mu \cdot dm \cdot g = \mu\frac{mg}{l}dx$。
4. 微元产生的阻力矩 $dM = x \cdot df = \mu\frac{mg}{l} x dx$。
5. 从 0 到 $l$ 积分：$M = \int_0^l \mu\frac{mg}{l} x dx = \mu\frac{mg}{l} [\frac{1}{2}x^2]_0^l = $ **$\frac{1}{2}\mu m g l$**。

#### 12. [计算题] 子弹斜打小球
**题目**：小球和细棒组成的系统对O轴的转动惯量为 $J$。质量 $m$ 速度 $v_0$ 的子弹，沿与水平面成 $\alpha$ 角方向射入并嵌在球心处。杆长 $l$。求共同角速度。
**图像说明**：摆锤悬挂，子弹从斜下角打入底部的球。
**解题步骤**：
1. **找核心套路**：绕固定轴碰撞，无外力矩，立刻上**角动量守恒**。
2. **算碰撞前角动量**：子弹的绝对速度是 $v_0$。把速度分解为水平 ($v_0\cos\alpha$) 和竖直 ($v_0\sin\alpha$)。只有水平分速度对O轴产生力矩（因为竖直分速度的延长线经过了转轴O点，力臂为0）。
   子弹对O的初始角动量 $L_0 = m(v_0\cos\alpha) \times l = mv_0l\cos\alpha$。
3. **算最终转动惯量**：子弹打进去后，子弹变成系统一部分，增加的转动惯量为 $ml^2$。
   最终总转动惯量 $J_{总} = J + ml^2$。
4. **列守恒等式求解**：
   $$L_0 = L_{最终} \implies mv_0l\cos\alpha = (J + ml^2)\omega$$
   解得最终角速度：
   $$\omega = \frac{mv_0l\cos\alpha}{J + ml^2}$$

#### 13. [计算题] 变力矩算时间
**题目**：圆盘($J$)初态 $\omega_0$，受阻力矩 $M = -k\omega$。求角速度变为 $\omega_0/2$ 所需时间。
**解题步骤**：
1. **列微分方程**：牛顿第二定律转动版：
   $$J\frac{d\omega}{dt} = -k\omega$$
2. **分离变量**：把 $\omega$ 移到左边，$dt$ 移到右边：
   $$\frac{d\omega}{\omega} = -\frac{k}{J}dt$$
3. **两边积分**：左边从 $\omega_0$ 积到 $\frac{1}{2}\omega_0$，右边从 $0$ 积到 $t$：
   $$\int_{\omega_0}^{\omega_0/2} \frac{1}{\omega} d\omega = \int_0^t -\frac{k}{J} dt$$
4. **计算对数**：
   $$[\ln \omega]_{\omega_0}^{\omega_0/2} = \ln(\frac{\omega_0/2}{\omega_0}) = \ln(\frac{1}{2}) = -\ln 2$$
   右侧积分结果为 $-\frac{k}{J}t$。
5. **得出结果**：
   $$-\ln 2 = -\frac{k}{J}t \implies t = \frac{J \ln 2}{k}$$

#### 14. [计算题] 双层滑轮系统
**题目**：两圆盘同轴粘合 ($m, r$ 和 $2m, 2r$)，对中心轴转动惯量 $J = \frac{9}{2}mr^2$。大圆盘和小圆盘边缘各挂一个质量为 $m$ 的重物。求角加速度的大小。
**图像说明**：同轴双圆盘，左边大盘边挂物块m，右边小盘边挂物块m。
**解题步骤**：
1. **判断转向**：左边挂在半径 $2r$ 处，右边挂在半径 $r$ 处。物块质量都是 $m$，显然左边力矩大，系统会逆时针转动。设角加速度大小为 $\alpha$。
2. **隔离列式**：
   * 左边物块向下加速：$mg - T_1 = ma_1$，且 $a_1 = \alpha(2r)$ $\implies T_1 = m(g - 2r\alpha)$。
   * 右边物块向上加速：$T_2 - mg = ma_2$，且 $a_2 = \alpha(r)$ $\implies T_2 = m(g + r\alpha)$。
3. **转动牛顿第二定律**：转盘受到的净力矩等于 $J\alpha$：
   $$T_1(2r) - T_2(r) = J\alpha = (\frac{9}{2}mr^2)\alpha$$
4. **代入求解**：
   $$[m(g - 2r\alpha)](2r) - [m(g + r\alpha)](r) = \frac{9}{2}mr^2\alpha$$
   展开合并同类项：
   $$2mgr - 4mr^2\alpha - mgr - mr^2\alpha = 4.5mr^2\alpha$$
   $$mgr - 5mr^2\alpha = 4.5mr^2\alpha$$
   $$mgr = 9.5mr^2\alpha = \frac{19}{2}mr^2\alpha$$
5. **最终结果**：
   $$\alpha = \frac{2g}{19r}$$

#### 15. [计算题] 面摩擦算刹车时间
**题目**：圆盘($m_0, R, J=\frac{1}{2}m_0R^2$)平放在有摩擦($\mu$)的桌面上。初静止。子弹($m, v_0$)打入边缘并留在盘上。求(1)打入后瞬间角速度 (2)圆盘多久后停下。
**图像说明**：水平圆盘，子弹沿切线打入。
**解题步骤**：
1. **第一问(角动量守恒)**：
   瞬间碰撞过程时间极短，摩擦力矩忽略不计。
   系统初角动量：$L_0 = mv_0R$。
   打入后总转动惯量：$J_{总} = J_{盘} + J_{弹} = \frac{1}{2}m_0R^2 + mR^2$。
   由 $L_0 = J_{总}\omega$ 得：
   $$\omega = \frac{mv_0R}{(\frac{1}{2}m_0 + m)R^2} = \frac{2mv_0}{(m_0 + 2m)R}$$
2. **第二问(求面摩擦阻力矩)**：
   题目说忽略子弹造成的摩擦力矩，只算圆盘的。这和第11题类似，要用到微积分套路（**必考基本功**）：
   在盘面上取半径 $r$ 处宽 $dr$ 的圆环，面积 $dS = 2\pi r dr$。
   微元质量 $dm = \frac{m_0}{\pi R^2} \cdot 2\pi r dr = \frac{2m_0}{R^2}r dr$。
   微元摩擦阻力矩 $dM_f = r \cdot (\mu dm \cdot g) = \frac{2\mu m_0 g}{R^2} r^2 dr$。
   盘的总摩擦力矩 $M_f = \int_0^R \frac{2\mu m_0 g}{R^2} r^2 dr = \frac{2}{3}\mu m_0 g R$。
3. **第三步(求停止时间)**：
   用角动量定理积分形式最快（无需再去算 $\alpha$）：阻力矩的冲量 = 角动量变化量
   $$M_f \cdot t = \Delta L$$
   系统初角动量为打入时的角动量 $\Delta L = L_0 = mv_0R$（刚好就是第一问算出来的）。
   $$(\frac{2}{3}\mu m_0 g R) \cdot t = mv_0R$$
   $$t = \frac{3mv_0}{2\mu m_0 g}$$

#### 16. [附加题] 人走动模型(相对运动理论)
**题目**：圆盘($J_{盘}=\frac{1}{2}m_{盘}R^2$, 质量令为 $m$)，人质量 $m_人 = m/10$。人站在 $R/2$ 处。初始系统以 $\omega_0$ 转动。人以相对圆盘的速率 $v$ 沿反方向走动。(1)求圆盘对地的绝对角速度。(2)要让圆盘相对地面静止，人要以多大速度往哪走？
**图像说明**：俯视图，圆盘转，人在上面逆向走。
**解题步骤**：
*   **绝对核心**：角动量守恒只看**相对地面的绝对速度(绝对角速度)**！
    设定初始转动方向（逆时针）为正方向。
1. **初态分析**：人和盘一起转。
   盘的惯量 $J_{盘} = \frac{1}{2}mR^2$。人的惯量 $J_人 = m_人(\frac{R}{2})^2 = (\frac{m}{10})(\frac{R^2}{4}) = \frac{1}{40}mR^2$。
   初始总角动量：$L_0 = (J_{盘} + J_人)\omega_0 = (\frac{1}{2} + \frac{1}{40})mR^2\omega_0 = \frac{21}{40}mR^2\omega_0$。
2. **第一问**：人往反方向走，设此时圆盘绝对角速度为 $\omega'$。
   人相对圆盘反方向走的角速度大小为 $\omega_{相对} = \frac{v}{R/2} = \frac{2v}{R}$。
   所以人的**绝对角速度**为 $\omega_人 = \omega' - \frac{2v}{R}$。
   新的系统角动量 $L_1 = J_{盘}\omega' + J_人\omega_人 = \frac{1}{2}mR^2\omega' + \frac{1}{40}mR^2(\omega' - \frac{2v}{R})$。
   由 $L_0 = L_1$，等式两边约掉 $mR^2$：
   $$\frac{21}{40}\omega_0 = \frac{21}{40}\omega' - \frac{2v}{40R}$$
   两边同乘40：$21\omega_0 = 21\omega' - \frac{2v}{R} \implies \omega' = \omega_0 + \frac{2v}{21R}$。
   *(这里结果大于 $\omega_0$，符合常理：你往反方向蹬圆盘，会给盘一个顺着原来转动方向的力，盘会转得更快)*。
3. **第二问**：要让盘相对地面静止，即 $\omega' = 0$。
   不管人在上面怎么走，系统不受外力矩，$L_{最终}$ 必须还是等于初态的 $\frac{21}{40}mR^2\omega_0$。
   盘停了($\omega'=0$)，这部分角动量全要靠人来提供：
   $$L_{最终} = J_人\omega_{人绝对} = \frac{1}{40}mR^2 \cdot \omega_{人绝对}$$
   所以：$\frac{1}{40}mR^2 \cdot \omega_{人绝对} = \frac{21}{40}mR^2\omega_0 \implies \omega_{人绝对} = 21\omega_0$。
   因为盘没转，人的绝对角速度就是相对圆盘的相对角速度。
   对应线速度大小为 $v = \omega_{人绝对} \times (\frac{R}{2}) = 21\omega_0 \times \frac{R}{2} = 10.5 R \omega_0$。
   **方向**：角动量是正的，说明人走的方向必须和**初始转盘转动方向相同**（顺着盘最初的方向死命跑，才能把转盘给“踩停”）。
