# 第四章：功与能

 * **动能定理：** 合外力做的功等于动能的变化量（W = \Delta E_k）。**套路**：遇到“变力做功”、“求初末速度”、或者“时间未知”的问题，首选动能定理。
 * **机械能守恒：** 只有重力或弹力做功时，系统的动能和势能相互转化，总和不变。**套路**：涉及高度变化、弹簧形变、光滑轨道，直接列等式 E_{k1} + E_{p1} = E_{k2} + E_{p2}。
 * **动量守恒（辅助）：** 某个方向上不受外力（通常是水平光滑面），该方向动量守恒。**套路**：多个物体相互作用（如滑块滑上半圆槽），用于寻找不同物体速度间的关系。
 * **角动量守恒（辅助）：** 质点受到的力始终指向某一点（有心力，如弹簧拉力、绳子拉力），则系统对该点的角动量守恒（mvr_{\perp} = \text{常数}）。**套路**：遇到光滑平面上的弹簧或绳子牵引圆周运动，必定要用。
## 一、 选择题详细讲解
### 1、变力做功积分
**题目：** 力 \vec{F} = F_0(x\vec{i} + y\vec{j}) 作用在质点上，质点从原点运动到 (0, 2R)，求做功。选项有 F_0R^2, 2F_0R^2, 3F_0R^2, 4F_0R^2。
**图像描述：** 质点在第一象限作圆周运动，路径从原点出发到达y轴上的 (0, 2R) 点。
**解题步骤：**
这道题核心是**变力做功的线积分**。功的定义式为 W = \int \vec{F} \cdot d\vec{r}。
 1. 写出微元位移 d\vec{r} = dx\vec{i} + dy\vec{j}。
 2. 计算点乘：\vec{F} \cdot d\vec{r} = F_0 x dx + F_0 y dy。
 3. 这是一个保守力场，做功与路径无关。我们可以直接沿坐标轴积分，或者直接看出它对应势能积分。
 4. 在终点 (0, 2R) 处：W = \int_0^0 F_0 x dx + \int_0^{2R} F_0 y dy = 0 + \left[ \frac{1}{2}F_0 y^2 \right]_0^{2R} = \frac{1}{2}F_0 (2R)^2 = 2F_0R^2。
   **答案：(B)**
### 2、动量与动能的关系
**题目：** 物体 A 和 B 质量和速率都不同，若 A 的动量数值比 B 大，比较动能 E_{KA} 与 E_{KB}。
**解题步骤：**
这是考察基本公式转换。
 1. 动能公式 E_k = \frac{1}{2}mv^2。
 2. 动量公式 p = mv。
 3. 联合推导：E_k = \frac{p^2}{2m}。
 4. 虽然已知 p_A > p_B，但质量 m_A 和 m_B 谁大谁小未知，因此无法比较比值的大小。
   **答案：(D)**
### 3、恒定阻力做功
**题目：** 速度为 v 的子弹打穿木板后速度为零，阻力恒定。求射入厚度一半时的速度。
**解题步骤：**
**套路：动能定理应用**。
 1. 设木板总厚度为 d，恒定阻力为 f。
 2. 全程动能定理：-fd = 0 - \frac{1}{2}mv^2。
 3. 一半深度的动能定理：-f(\frac{d}{2}) = \frac{1}{2}mv'^2 - \frac{1}{2}mv^2。
 4. 将上式代入下式：-\frac{1}{2}(\frac{1}{2}mv^2) = \frac{1}{2}mv'^2 - \frac{1}{2}mv^2 \implies \frac{1}{2}mv'^2 = \frac{1}{4}mv^2 \implies v'^2 = \frac{1}{2}v^2。得出 v' = \frac{1}{\sqrt{2}}v。
   **答案：(D)**
### 4、空间恒力做功点乘
**题目：** 位移 \Delta\vec{r} = 4\vec{i} - 5\vec{j} + 6\vec{k}，恒力 \vec{F} = -3\vec{i} - 5\vec{j} + 9\vec{k}，求做功。
**解题步骤：**
**套路：恒力做功直接点乘**。
 1. 功 W = \vec{F} \cdot \Delta\vec{r}。
 2. 对应坐标相乘并相加：W = (-3)(4) + (-5)(-5) + (9)(6) = -12 + 25 + 54 = 67 \text{ J}。
   **答案：(C)**
### 5、运动学结合动能定理
**题目：** m=0.5\text{kg}，运动方程 x=5t, y=0.5t^2，求 t=2\text{s} 到 4\text{s} 外力做的功。
**解题步骤：**
**套路：先求导找速度，再用动能定理求功**。
 1. 对位移求导得速度分量：v_x = 5， v_y = t。速度平方 v^2 = v_x^2 + v_y^2 = 25 + t^2。
 2. 计算 t=2 时的动能：v_2^2 = 25 + 4 = 29。
 3. 计算 t=4 时的动能：v_4^2 = 25 + 16 = 41。
 4. 动能定理 W = \Delta E_k = \frac{1}{2}m(v_4^2 - v_2^2) = \frac{1}{2}(0.5)(41 - 29) = \frac{1}{4} \times 12 = 3 \text{ J}。
   **答案：(B)**
### 6、弹性势能与动能转化
**题目：** 非线性弹簧弹性力 F=kx^3，滑块初速度为 v，求最大压缩长度。
**解题步骤：**
**套路：变力势能积分 + 机械能守恒**。
 1. 变力产生的弹性势能是力的积分：E_p = \int_0^x kx^3 dx = \frac{1}{4}kx^4。
 2. 压缩到最大长度时，动能全部转化为势能：\frac{1}{2}mv^2 = \frac{1}{4}kx^4。
 3. 求解 x：x^4 = \frac{2mv^2}{k} \implies x = \left(\frac{2mv^2}{k}\right)^{\frac{1}{4}}。
   **答案：(D)**
## 二、 填空题详细讲解
### 7、万有引力做功
**题目：** 火箭质量 m，从地球表面上升到距地面 2R 高处，求地球引力做的功。
**解题步骤：**
**套路：万有引力公式积分**。
 1. 地球引力 F = -G\frac{M_e m}{r^2}。注意起始位置到地心距离为 R，末位置到地心距离为 R + 2R = 3R。
 2. 积分求功：W = \int_{R}^{3R} -G\frac{M_e m}{r^2} dr = G M_e m \left[ \frac{1}{r} \right]_R^{3R} = G M_e m (\frac{1}{3R} - \frac{1}{R}) = -\frac{2 G M_e m}{3R}。
   *(注：如果题目允许用地面重力加速度 g = \frac{GM_e}{R^2} 代换，也可写成 -\frac{2mgR}{3})*。
### 8、圆周运动中的恒力做功
**题目：** 质点逆时针走过 3/4 圆周，从 A 到 B，恒力 \vec{F}_0 = F_0\vec{i} 做功多少。
**解题步骤：**
**套路：恒力做功，只看位移**。
由于 \vec{F}_0 始终沿 x 轴正向，不管路径多么复杂，做功只和 x 方向的位移有关：W = F_0 \Delta x。
 1. 观察配图，A 点位于 y 轴负半轴底部（坐标为 (0, -R)）。
 2. 逆时针走过 3/4 圆周，B 点位于 x 轴负半轴左侧（坐标为 (-R, 0)）。
 3. x 方向的位移 \Delta x = x_B - x_A = -R - 0 = -R。
 4. W = F_0 \times (-R) = -F_0 R。
### 9、冲量与做功综合
**题目：** 质量 1.0\text{kg}，运动方程 x=3t-4t^2+t^3，求 0 到 4s 的冲量和做功。
**解题步骤：**
 1. 求导得速度：v = \frac{dx}{dt} = 3 - 8t + 3t^2。
 2. 代入时间求初末速度：t=0 时 v_0 = 3\text{ m/s}；t=4 时 v_4 = 3 - 32 + 48 = 19\text{ m/s}。
 3. **冲量等于动量变化量**：I = \Delta p = m(v_4 - v_0) = 1.0 \times (19 - 3) = 16\text{ N}\cdot\text{s}。
 4. **做功等于动能变化量**：W = \Delta E_k = \frac{1}{2}m(v_4^2 - v_0^2) = 0.5 \times (19^2 - 3^2) = 0.5 \times (361 - 9) = 176\text{ J}。
### 10、有心力场机械能
**题目：** 平方反比力 F=-k/r^2 作用下的圆周运动，求速度和总机械能。
**解题步骤：**
 1. **向心力公式**：F_c = m\frac{v^2}{r} = \frac{k}{r^2} \implies v = \sqrt{\frac{k}{mr}}。
 2. 动能：E_k = \frac{1}{2}mv^2 = \frac{k}{2r}。
 3. 势能（引力场套路积分）：E_p = \int_{\infty}^r (-F) dr = \int_{\infty}^r \frac{k}{r^2} dr = -\frac{k}{r}。
 4. 总机械能：E = E_k + E_p = \frac{k}{2r} - \frac{k}{r} = -\frac{k}{2r}。
### 11、角动量守恒与能量守恒结合
**题目：** 光滑水平面弹簧栓物体，给一初速度 v_0，当速率降为 \frac{1}{2}v_0 时求弹簧拉力。
**解题步骤：**
这是一个极经典的考法，必须同时列出两个守恒定律：
 1. **角动量守恒**：弹力始终指向固定点 O（有心力），故角动量守恒。初态垂直，角动量 L_0 = mv_0 l_0。
 2. **机械能守恒**：系统只有弹簧做功，初机械能 E_0 = \frac{1}{2}mv_0^2。
 3. 当速率为 v = \frac{1}{2}v_0 时，动能变为 \frac{1}{2}m(\frac{1}{2}v_0)^2 = \frac{1}{8}mv_0^2。
 4. 损失的动能转化为弹性势能：\Delta E_p = \frac{1}{2}mv_0^2 - \frac{1}{8}mv_0^2 = \frac{3}{8}mv_0^2。
 5. 由弹性势能公式 \frac{1}{2}k(\Delta x)^2 = \frac{3}{8}mv_0^2 \implies \Delta x = v_0\sqrt{\frac{3m}{4k}}。
 6. 拉力 f = k\Delta x = k v_0\sqrt{\frac{3m}{4k}} = \frac{1}{2}v_0\sqrt{3mk}。

### 17、半圆槽内的相对与绝对运动
**题目：** 小物体从质量为 M 的光滑半圆槽边缘 A 滑到最低点 B，求速度和圆槽移动距离。
**解题步骤：**
**套路：水平动量守恒 + 机械能守恒 + 质心位移公式**。
 1. **求速度**：
   滑到最低点 B 时，设小物体对地速度为 v（向右），槽对地速度为 V（向左）。
   * 水平动量守恒：mv = MV \implies V = \frac{m}{M}v。
   * 机械能守恒：mgR = \frac{1}{2}mv^2 + \frac{1}{2}MV^2。
   * 联立求解得小物体对地速度：v = \sqrt{\frac{2gRM}{M+m}}。
   * 槽对地速度：V = m\sqrt{\frac{2gR}{M(M+m)}}。
   * 小物体对槽的相对速度等于两者反向速度的标量和：v_{\text{相对}} = v + V = \sqrt{\frac{2gR(M+m)}{M}}。
 2. **求槽移动距离**：
   这是一个常见结论“人船模型”的直接应用。系统在水平方向质心不动。
   设槽移动了 \Delta X，滑块水平位移为 R - \Delta X。
