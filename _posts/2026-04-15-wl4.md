## 变力功：找准积分变量

如果力不是常数（随位置变），那就必须积分 。

- **直角坐标系**：把力 $\vec{F}$ 拆成 $F_x, F_y$，位移 $d\vec{r}$ 拆成 $dx, dy$。
    
    - 例如第1题：力是 $F_x = F_0 x, F_y = F_0 y$ 。做功就是分别对 $x$ 和 $y$ 积分：$\int F_0 x dx + \int F_0 y dy$ 。
        
- **路径无关性**：对于保守力（如引力、弹力），功只看起点和终点，中间怎么走的圆弧还是直线，不影响结果 。
    

## 动能定理：前后的“快慢”对比

当你看到题目里提到**速度变化**、**位移**、**阻力**，直接反应动能定理 。

- **子弹打木板模型**（第3题）：阻力做的功等于动能的改变量 。
    
    - 打穿全厚度 $d$：$-fd = 0 - \frac{1}{2}mv^2$ 。
        
    - 打到一半 $d/2$：$-f(d/2) = \frac{1}{2}m(v')^2 - \frac{1}{2}mv^2$ 。
        
    - **技巧**：两式相比，直接消掉多余的 $f, d, m$ 。
        

## 守恒定律：寻找系统的“不动量”

物理题最爱考的就是在乱动之中，有什么东西是**没变**的。

- **角动量守恒**（第14、15题）：
    
    - **特征**：物体绕着中心转，半径在变，但没受到绕中心转动的力矩 。
        
    - **公式**：$m v_1 r_1 = m v_2 r_2 \sin\theta$ 。
        
- **水平动量守恒**（第16、17题）：
    
    - **特征**：一个物体在另一个物体上滑，地面光滑，整体在水平方向没受外力 。
        
    - **公式**：$m v_x + M V_X = 0$ 。
        

## 势能与机械能：无损转换

如果系统里只有重力、弹力或万有引力在干活，总能量（动能 + 势能）就是个定值 。

- **引力做功与势能**：
    
    - 从地面上升到 $2R$ 高处（即距离地心 $3R$），引力做负功 。
        
    - 计算公式：$A = \int_{R}^{3R} -\frac{G M m}{r^2} dr$ 。
        
- **特殊弹簧**（第6题）：
    
    - 如果力不是 $kx$ 而是 $kx^3$，它的势能就不是 $\frac{1}{2}kx^2$，而是通过积分 $\int kx^3 dx$ 得到的 $\frac{1}{4}kx^4$ 。
        

## 质心运动定理：解决位移难题

附加题（第17题第2问）这种问“底座移动了多少距离”的题，不需要算复杂的加速度 。

- **逻辑**：系统水平方向动量守恒且初始静止，说明系统的**质心水平位置不动** 。
    
- **简便公式**：$m \Delta x = M \Delta X$。物体往左走多少，底座就按质量比例往右补多少 。

# 🎯 例题

## 第1题
一质点在坐标平面内作圆周运动，有一力 $\vec{F}=F_{0}(x\vec{i}+y\vec{j})$ (SI) 作用在质点上，在该质点从坐标原点运动到 $(0,2R)$ 位置过程中，力 $\vec{F}$ 对它所作的功为 
(A) $F_{0}R^{2}$ (B) $2F_{0}R^{2}$ (C) $3F_{0}R^{2}$ (D) $4F_{0}R^{2}$ 

**图像说明**：质点沿圆弧从原点 $(0,0)$ 移动到 $y$ 轴上的 $(0,2R)$。

**解题步骤**：
直接对分量积分：
$$A = \int_{0}^{0} F_0 x dx + \int_{0}^{2R} F_0 y dy$$
由于 $x$ 的积分区间是 $0$ 到 $0$，该项为 $0$ 。
$$A = 0 + \left[ \frac{1}{2} F_0 y^2 \right]_{0}^{2R} = \frac{1}{2} F_0 (2R)^2 = 2F_0 R^2$$
选 **B** 。

## 第2题
已知两个物体A和B的质量以及它们的速率都不相同，若物体A的动量在数值上比物体B的大，则A的动能 $E_{KA}$ 与B的动能 $E_{KB}$ 之间 
(A) $E_{KB}$ 大于 $E_{KA}$ (B) $E_{KB}$ 小于 $E_{KA}$ (C) $E_{KB}=E_{KA}$ (D) 不能判定 

**解题步骤**：
根据 $E_k = \frac{P^2}{2m}$ 。
虽然 $P_A > P_B$，但因为质量 $m_A$ 和 $m_B$ 的大小关系未知，分子大不代表数值一定大 。
选 **D** 。

## 第3题
速度为 $v$ 的子弹，打穿一块不动的木板后速度变为零，设木板对子弹的阻力是恒定的。那么，当子弹射入木板的深度等于其厚度的一半时，子弹的速度是 
(A) $\frac{1}{4}v$ (B) $\frac{1}{3}v$ (C) $\frac{1}{2}v$ (D) $\frac{1}{\sqrt{2}}v$ 

**解题步骤**：
设木板厚度为 $d$，阻力为 $f$。
1. 全程：$-fd = 0 - \frac{1}{2}mv^2$ 。
2. 半程：$-f\frac{d}{2} = \frac{1}{2}mv'^2 - \frac{1}{2}mv^2$ 。
将 1 式代入 2 式：$\frac{1}{2}( - \frac{1}{2}mv^2) = \frac{1}{2}mv'^2 - \frac{1}{2}mv^2$。
整理得：$\frac{1}{2}mv'^2 = \frac{1}{4}mv^2 \Rightarrow v' = \frac{v}{\sqrt{2}}$ 。
选 **D** 。

## 第4题
一个质点位移为 $\Delta\vec{r}=4\vec{i}-5\vec{j}+6\vec{k}$ (SI)，其中一个力为恒力 $\vec{F}=-3\vec{i}-5\vec{j}+9\vec{k}$ (SI)，则此力在该位移过程中所作的功为 
(A) -67 J (B) 17 J (C) 67 J (D) 91 J 

**解题步骤**：
恒力做功直接求点积：
$$A = \vec{F} \cdot \Delta\vec{r} = (-3)(4) + (-5)(-5) + (9)(6)$$
$$A = -12 + 25 + 54 = 67 \text{ J}$$
选 **C** 。

## 第5题
质量为 $m=0.5\text{kg}$ 的质点，运动方程为 $x=5t, y=0.5t^2$ (SI)，从 $t=2\text{s}$ 到 $4\text{s}$ 这段时间内，外力对质点作的功为 
(A) 1.5 J (B) 3 J (C) 4.5 J (D) -1.5 J 

**解题步骤**：
1. 求速度分量：$v_x = \frac{dx}{dt} = 5$，$v_y = \frac{dy}{dt} = t$ 。
2. 动能公式：$E_k = \frac{1}{2} \times 0.5 \times (5^2 + t^2) = 0.25(25 + t^2)$ 。
3. 动能定理：$W = E_k(4) - E_k(2)$ 。
$$W = 0.25(25 + 16) - 0.25(25 + 4) = 0.25(41 - 29) = 3 \text{ J}$$
选 **B** 。

## 第6题
特殊的轻弹簧弹性力 $F=kx^3$，压缩弹簧，则弹簧被压缩的最大长度为 
(A) $\sqrt{\frac{m}{k}}v$ (B) $\sqrt{\frac{k}{m}}v$ (C) $(\frac{4mv}{k})^{\frac{1}{4}}$ (D) $(\frac{2mv^2}{k})^{\frac{1}{4}}$ 

**解题步骤**：
机械能守恒，初始动能全部转化为弹簧势能（变力功）：
$$\frac{1}{2}mv^2 = \int_{0}^{x_{max}} kx^3 dx$$
$$\frac{1}{2}mv^2 = \frac{1}{4}kx_{max}^4$$
解得：$x_{max} = (\frac{2mv^2}{k})^{\frac{1}{4}}$ 。
选 **D** 。

## 第7题
地球质量为 $m_e$，半径为 $R$。一质量为 $m$ 的火箭从地面上升到距地面高度为 $2R$ 处，在此过程中，地球引力对火箭作的功为 

**解题步骤**：
引力做功等于引力势能变化的负值。末位置距离地心 $r_2 = R + 2R = 3R$ 。
$$A = -\Delta E_p = - \left( -\frac{Gm_em}{3R} - (-\frac{Gm_em}{R}) \right)$$
$$A = \frac{Gm_em}{3R} - \frac{Gm_em}{R} = -\frac{2Gm_em}{3R}$$
填入：$-\frac{2Gm_em}{3R}$ 。

## 第8题
半径为 $R$ 圆周运动，恒力 $\vec{F}_0 = F_0\vec{i}$。从 $A$ 点逆时针走过 $3/4$ 圆周到 $B$ 点，力所作的功为 

**图像说明**：$A$ 点在 $(R,0)$，$B$ 点在 $(0,R)$。

**解题步骤**：
恒力功只看位移矢量 $\Delta\vec{r} = \vec{r}_B - \vec{r}_A = (0-R)\vec{i} + (R-0)\vec{j} = -R\vec{i} + R\vec{j}$ 。
$$A = \vec{F}_0 \cdot \Delta\vec{r} = (F_0\vec{i}) \cdot (-R\vec{i} + R\vec{j}) = -F_0R$$
填入：$-F_0R$ 。

## 第9题
$m=1.0\text{kg}$，运动方程 $x=3t-4t^2+t^3$ (SI)。在 $0$ 到 $4\text{s}$ 内：(1) 冲量 $I=$ (2) 功 $W=$ 

**解题步骤**：
$v = \frac{dx}{dt} = 3 - 8t + 3t^2$ 。
$t=0$ 时，$v_0 = 3\text{m/s}$；$t=4$ 时，$v_4 = 3 - 32 + 48 = 19\text{m/s}$ 。
1. 冲量：$I = m\Delta v = 1.0 \times (19 - 3) = 16 \text{ N}\cdot\text{s}$ 。
2. 功：$W = \Delta E_k = \frac{1}{2} \times 1.0 \times (19^2 - 3^2) = \frac{1}{2}(361 - 9) = 176 \text{ J}$ 。

## 第10题
质点在 $F=-k/r^2$ 作用下作半径为 $r$ 的圆周运动。求速度 $v$ 和机械能 $E$（零点在无穷远） 

**解题步骤**：
1. 向心力公式：$\frac{k}{r^2} = m\frac{v^2}{r} \Rightarrow v = \sqrt{\frac{k}{mr}}$ 。
2. 机械能：$E = E_k + E_p = \frac{1}{2}mv^2 + (-\frac{k}{r})$ 。
$$E = \frac{1}{2}m(\frac{k}{mr}) - \frac{k}{r} = \frac{k}{2r} - \frac{k}{r} = -\frac{k}{2r}$$
填入：$\sqrt{k/(mr)}$ 和 $-k/(2r)$ 。

## 第11题
弹簧 $k$，物体 $m$，初速 $v_0$ 垂直弹簧。当速率为 $\frac{1}{2}v_0$ 时弹簧拉力 $f=$ 

**解题步骤**：
机械能守恒求伸长量 $x$ ：
$$\frac{1}{2}mv_0^2 = \frac{1}{2}m(\frac{1}{2}v_0)^2 + \frac{1}{2}kx^2$$
$$\frac{1}{2}mv_0^2 = \frac{1}{8}mv_0^2 + \frac{1}{2}kx^2 \Rightarrow \frac{3}{8}mv_0^2 = \frac{1}{2}kx^2 \Rightarrow x = \sqrt{\frac{3mv_0^2}{4k}}$$
拉力 $f = kx = k \cdot \frac{v_0}{2} \sqrt{\frac{3m}{k}} = \frac{v_0}{2} \sqrt{3km}$ 。

## 第12题
提水桶，深 $10\text{m}$，初始共重 $11\text{kg}$。每升高 $1\text{m}$ 漏水 $0.2\text{kg}$。求人作的功 

**解题步骤**：
总质量随高度 $h$ 变化：$M(h) = (10+1) - 0.2h = 11 - 0.2h$ 。
人提桶的力 $F = M(h)g$（匀速）。
$$A = \int_{0}^{10} (11 - 0.2h) \times 9.8 dh = 9.8 \times [11h - 0.1h^2]_{0}^{10}$$
$$A = 9.8 \times (110 - 10) = 980 \text{ J}$$

## 第13题
陨石高 $h$ 处速度 $v_0$，求落地速度 $v$ 

**解题步骤**：
直接用机械能守恒：
$$\frac{1}{2}mv_0^2 - \frac{GMm}{R+h} = \frac{1}{2}mv^2 - \frac{GMm}{R}$$
$$\frac{1}{2}mv^2 = \frac{1}{2}mv_0^2 + GMm(\frac{1}{R} - \frac{1}{R+h}) = \frac{1}{2}mv_0^2 + \frac{GMmh}{R(R+h)}$$
$$v = \sqrt{v_0^2 + \frac{2GMh}{R(R+h)}}$$

## 第14题
物体 $A$ 距圆心 $r_0$，角速度 $\omega_0$。下拉绳子使半径变为 $\frac{1}{2}r_0$，求功 $W$ 

**解题步骤**：
1. 角动量守恒求横向速度 $v'$ ：
$mr_0(r_0\omega_0) = m(\frac{1}{2}r_0)v' \Rightarrow v' = 2r_0\omega_0$ 。
2. 此时总速度平方 $v_{总}^2 = v'^2 + v_{拉}^2$ 。
3. 动能定理：$W = \frac{1}{2}m(v_{总}^2) - \frac{1}{2}m(r_0\omega_0)^2$ 。
$$W = \frac{1}{2}m(4r_0^2\omega_0^2 + v_{拉}^2) - \frac{1}{2}mr_0^2\omega_0^2 = \frac{3}{2}mr_0^2\omega_0^2 + \frac{1}{2}mv_{拉}^2$$

## 第15题
$m=1\text{kg}$，弹簧原长 $0.2\text{m}$，$k=100$。$t=0$ 时长度 $0.2\text{m}$，$v_0=5\text{m/s}$ 垂直弹簧。当长度 $l=0.5\text{m}$ 时求速度 $v$ 和夹角 $\theta$ 

**解题步骤**：
1. 机械能守恒求 $v$ ：
$\frac{1}{2}mv_0^2 = \frac{1}{2}mv^2 + \frac{1}{2}k(l-l_0)^2$ 。
$25 = v^2 + 100(0.3)^2 \Rightarrow v^2 = 25 - 9 = 16 \Rightarrow v = 4 \text{ m/s}$ 。
2. 角动量守恒求 $\theta$ ：
$mv_0l_0 = mvl\sin\theta \Rightarrow 5 \times 0.2 = 4 \times 0.5 \times \sin\theta$
$1 = 2\sin\theta \Rightarrow \sin\theta = 0.5 \Rightarrow \theta = 30^\circ$ 。

## 第17题
半圆槽 $M$、滑块 $m$，从 $A$ 点静止下滑。求 (1) $B$ 点时的速度 (2) 槽移动距离 

**解题步骤**：
1. 速度：系统水平动量守恒 $m(v-V)-MV=0$ 和机械能守恒 $\frac{1}{2}m(v-V)^2 + \frac{1}{2}MV^2 = mgR$ 。
解得槽速 $V = m\sqrt{\frac{2gR}{M(m+M)}}$ 。
2. 位移：利用质心在水平方向不动（因为水平合外力为 $0$）。
$m(R-S) = MS \Rightarrow S = \frac{mR}{m+M}$ 。
