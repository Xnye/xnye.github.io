### 1. 添加后缀的规则

| 单词末尾类型 | 判定条件 (后缀开头) | 动作 | 典型示例 |
| :--- | :--- | :--- | :--- |
| **辅音 + 不发音 e** | 加 **辅音**后缀 (-ly, -ment, -ful) | **-** | complete → **completely** |
| | 加 **元音**后缀 (-ing, -ed, -able) | **去 e** | move → **moving** |
| **辅音 + y** | - | **y 变 i** | happy → **happily** |
| **元音 + y** | - | **-** | play → **played** |
| **-le 结尾** | 加 **-ly** 时 | **去 e 加 y** | simple → **simply** |
| **-ue 结尾** | 加 **-ly** 时 | **去 e 加 ly** | true → **truly** |
| **-ie 结尾** | 加 **-ing** 时 | **ie 变 y** | die → **dying** |
| **1-1-1结构**短单词 | 加 **元音**后缀 (-ing, -ed, -er) | **双写末尾辅音** | run → **running** |
| **多音节重读在后** | 加 **元音**后缀 (末尾符合1-1) | **双写末尾辅音** | refer → **referred** |

最后两条可以这么简化：

**单词短吗？** 短且只有一个元音 → 双写（hit → hitting）。

长得厚实（元音多）→ 直接加（clean → cleaning）。

**单词长吗？** 重音在最后 → 双写（admit → admitting）。

重音在开头 → 不双写（happen → happening）。
