## 9-1：什么是多元函数？

**一元函数**：$y = f(x)$，一个输入，一个输出
**多元函数**：$z = f(x, y)$，**两个输入**，一个输出

> 类比：温度取决于**经度和纬度**，不是单一变量。$T = f(x, y)$

## 概念一：函数表达式 & 复合

**题目(1)** 给你 $f(x,y) = 2x + y^2$，求 $f[xy,\ f(x,y)]$

**思路**：把括号里的东西**机械地塞进去**

$$f(\textcolor{red}{u}, \textcolor{blue}{v}) = 2\textcolor{red}{u} + \textcolor{blue}{v}^2$$

这里 $\textcolor{red}{u} = xy$，$\textcolor{blue}{v} = f(x,y) = 2x+y^2$

$$f[xy, f(x,y)] = 2(xy) + (2x+y^2)^2$$

---

## 概念二：定义域

函数有意义的 $(x,y)$ 的范围。

**要记住的限制条件：**

| 结构 | 限制 |
|------|------|
| $\sqrt{A}$ | $A \geq 0$ |
| $\ln A$ | $A > 0$ |
| $\frac{1}{A}$ | $A \neq 0$ |

---

## 概念三：二重极限

$(x,y) \to (a,b)$ 时，函数趋近某个值。

**关键陷阱**：二元极限要求**从任意方向趋近**结果都一样，否则极限不存在！

**常用技巧：**
- 等价无穷小：$\sin u \approx u$，$1-\cos u \approx \frac{u^2}{2}$（当 $u\to 0$）
- 重要极限：$\lim_{t\to 0}(1+t)^{1/t} = e$

# 🎯 例题

## 题1(1) — 复合函数

已知 $f(x,y) = 2x + y^2$，求 $f[xy,\ f(x,y)]$

**方法**：第一个位置放 $xy$，第二个位置放 $f(x,y)$

$$= 2(xy) + (2x+y^2)^2$$

$$\boxed{= 2xy + (2x+y^2)^2}$$

---

## 题1(2) — 求定义域

$$z = \frac{\sqrt{x-y^2}}{\ln(1-x^2-y^2)}$$

**两个限制同时满足：**

- $\sqrt{x-y^2}$ 要求：$x - y^2 \geq 0 \Rightarrow x \geq y^2$
- $\ln(1-x^2-y^2)$ 要求：$1-x^2-y^2 > 0 \Rightarrow x^2+y^2 < 1$
- 分母不为零：$\ln(\cdot) \neq 0 \Rightarrow 1-x^2-y^2 \neq 1 \Rightarrow x^2+y^2 \neq 0$（即不能是原点，但原点已被第一个条件排除）

$$\boxed{\{(x,y)\ |\ x \geq y^2 \text{ 且 } 0 < x^2+y^2 < 1\}}$$

---

## 题1(3) — 极限（等价无穷小）

$$\lim_{(x,y)\to(0,0)} \frac{\sin(xe^y)}{x}$$

当 $(x,y)\to(0,0)$ 时，$xe^y \to 0$，所以 $\sin(xe^y) \approx xe^y$

$$= \lim_{(x,y)\to(0,0)} \frac{xe^y}{x} = \lim e^y = e^0 = \boxed{1}$$

---

## 题1(4) — 间断点

$$z = \frac{1}{x^2 + xy}$$

分母为零的地方就是间断点：

$$x^2 + xy = 0 \Rightarrow x(x+y) = 0$$

$$\Rightarrow x = 0 \text{ 或 } x = -y$$

$$\boxed{\{(x,y)\ |\ x=0 \text{ 或 } x=-y\}}$$

---

## 题2 — 反解函数

已知 $f\!\left(x+y,\ \dfrac{y}{x}\right) = x^2 - y^2$，求 $f(x,y)$

**技巧：换元法**，令 $a = x+y$，$b = \dfrac{y}{x}$，反解出 $x, y$

从 $b = y/x$ 得 $y = bx$，代入 $a = x+y = x+bx = x(1+b)$

$$x = \frac{a}{1+b},\quad y = \frac{ab}{1+b}$$

$$x^2 - y^2 = (x+y)(x-y) = a \cdot \left(\frac{a}{1+b} - \frac{ab}{1+b}\right) = a \cdot \frac{a(1-b)}{1+b}$$

$$\boxed{f(x,y) = x^2 \cdot \frac{1-y}{1+y} = \frac{x^2(1-y)}{1+y}}$$

---

## 题3 — 由已知条件求函数

已知 $z = \sqrt{y} + f(\sqrt{x}-1)$，当 $y=1$ 时 $z=x$，求 $f(x)$

**第一步**：代入 $y=1$：

$$x = 1 + f(\sqrt{x}-1)$$
$$f(\sqrt{x}-1) = x - 1$$

**第二步**：换元，令 $k = \sqrt{x}-1$，则 $\sqrt{x} = k+1$，即 $x = (k+1)^2$

$$f(k) = (k+1)^2 - 1 = k^2 + 2k$$

$$\boxed{f(x) = x^2 + 2x}$$

验证结果：
$$z = \sqrt{y} + f(\sqrt{x}-1) = \sqrt{y} + (\sqrt{x}-1)^2 + 2(\sqrt{x}-1) = \sqrt{y} + x - 1 = x + \sqrt{y} - 1$$

---

## 题4(1) — 极限计算

$$\lim_{(x,y)\to(0,0)} \frac{1-\cos(xy)}{xy^2}$$

用等价无穷小：$1-\cos(xy) \approx \dfrac{(xy)^2}{2}$

$$= \lim \frac{\frac{(xy)^2}{2}}{xy^2} = \lim \frac{x^2y^2}{2xy^2} = \lim \frac{x}{2} = \boxed{0}$$

---

## 题4(2) — $1^\infty$ 型极限

$$\lim_{(x,y)\to(0,0)} (1 + x\cos y)^{1/x}$$

当 $(x,y)\to(0,0)$：$\cos y \to 1$，$x\cos y \to 0$，这是 $1^\infty$ 型

用公式 $\lim(1+\alpha)^{1/\alpha} = e$ 当 $\alpha \to 0$：

$$= \lim e^{\frac{1}{x} \cdot x\cos y} = e^{\cos y}\big\rvert_{y\to 0} = \boxed{e^1 = e}$$

---

## 题4(3) — 无穷远处极限

$$\lim_{(x,y)\to(+\infty,+\infty)} \left(\frac{xy}{x^2+y^2}\right)^{xy}$$

**关键不等式**：由均值不等式 $x^2 + y^2 \geq 2xy$，所以

$$\frac{xy}{x^2+y^2} \leq \frac{1}{2}$$

且当 $x,y\to+\infty$ 时，令 $t = xy \to +\infty$，底数 $\leq 1/2$，指数 $\to +\infty$：

$$\left(\frac{1}{2}\right)^{+\infty} = 0, \quad \text{更何况底数} < \frac{1}{2}$$

$$\boxed{= 0}$$

---

## 题5(1) — 证明极限不存在

$$\lim_{(x,y)\to(0,0)} \frac{\sin(x-y)}{x+y}$$

**方法：走不同路径，得到不同极限值，则极限不存在**

**路径1**：$y = 0$，$x\to 0$

$$\frac{\sin x}{x} \to 1$$

**路径2**：$x = 0$，$y\to 0$

$$\frac{\sin(-y)}{y} = \frac{-\sin y}{y} \to -1$$

两条路径极限值不同（$1 \neq -1$），∴ **极限不存在** ✓

---

## 题5(2) — 证明极限不存在

$$\lim_{(x,y)\to(0,0)} \frac{x^4+y^2}{x^2+y^4}$$

**路径1**：$y = 0$，$x\to 0$

$$\frac{x^4}{x^2} = x^2 \to 0$$

**路径2**：$x = 0$，$y\to 0$

$$\frac{y^2}{y^4} = \frac{1}{y^2} \to +\infty$$

极限值不同，∴ **极限不存在** ✓

---

## 🗂️ 总结

| 题型 | 核心方法 |
|------|----------|
| 复合函数 | 机械代入，位置对应 |
| 极限（$\to 0$）| 等价无穷小替换 |
| 极限（$1^\infty$）| $e^{\text{指数}\times\text{底数}-1}$ |
| 极限（$\to\infty$）| 放缩+夹逼 |
| 证明不存在 | 两条路径结果不同 |
| 反解函数 | 换元，解方程组 |


# 9-2：偏导数

## 🧠 偏导数是什么？

对多元函数求导时，**把其他变量当常数，只对一个变量求导**。

> 类比：你在山上，偏导数 $\frac{\partial z}{\partial x}$ = 只往东西方向走时的坡度；$\frac{\partial z}{\partial y}$ = 只往南北方向走时的坡度。

---

## 📋 公式

| 函数 | 对x求导 |
|------|---------|
| $x^n$ | $nx^{n-1}$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\frac{1}{\cos^2 x}$ |
| $\arctan x$ | $\frac{1}{1+x^2}$ |
| $\sqrt{x}$ | $\frac{1}{2\sqrt{x}}$ |

**链式法则**（复合函数）：$[\,f(g(x))\,]' = f'(g(x)) \cdot g'(x)$

---

# 🎯 逐题讲解

---

### 1(1) — 偏导数定义

**偏导数的正确定义**：

$$f_x(x_0, y_0) = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x,\ \textcolor{red}{y_0}) - f(x_0,\ \textcolor{red}{y_0})}{\Delta x}$$

**关键**：y 必须固定在 $y_0$，不能变！

- A：$y$ 变成了 $y+\Delta y$，❌
- B：$y$ 固定在 $y_0$，分子分母都对 ✅
- C：分子里 $f(x_0,y)$，$y$ 没有固定 ❌
- D：分子里 $f(x_0, y)$，$y$ 没有固定 ❌

$$\boxed{答案：B}$$

---

### 1(2) — 极限变形识别

已知 $f_y(a,b) = 2$，求：

$$\lim_{\Delta y \to 0} \frac{f(a,\ b+\Delta y) - f(a,\ b-\Delta y)}{\Delta y}$$

**技巧**：把这个极限拆开凑成偏导数定义形式。

分子加减 $f(a,b)$：

$$= \lim_{\Delta y\to 0} \frac{f(a,b+\Delta y)-f(a,b)}{\Delta y} + \lim_{\Delta y\to 0}\frac{f(a,b)-f(a,b-\Delta y)}{\Delta y}$$

第一项 $= f_y(a,b) = 2$

第二项：令 $t = -\Delta y$，当 $\Delta y\to 0$ 时 $t\to 0$：

$$\frac{f(a,b)-f(a,b+t)}{-t} \cdot \frac{-t}{\Delta y}\bigg|_{t=-\Delta y} = f_y(a,b) = 2$$

所以结果 $= 2 + 2 = $

$$\boxed{答案：C（4）}$$

---

### 1(3) — 用定义求偏导（非光滑点）

$f(x,y) = \sqrt[3]{xy}$，求 $f_x(0,0)$

**不能直接用求导公式**（因为在原点处函数不光滑），必须**回到定义**：

$$f_x(0,0) = \lim_{\Delta x \to 0} \frac{f(0+\Delta x,\ 0) - f(0,0)}{\Delta x}$$

$$= \lim_{\Delta x \to 0} \frac{\sqrt[3]{\Delta x \cdot 0} - 0}{\Delta x} = \lim_{\Delta x \to 0} \frac{0}{\Delta x} = 0$$

$$\boxed{答案：A（0）}$$

---

### 1(4) — 代入特殊值再求导

$f(x,y) = x + (y-1)\sin\sqrt{\dfrac{x}{y}}$，求 $f_x(x, 1)$

**技巧：先代入 $y=1$，再对 $x$ 求导**（因为求的是 $y=1$ 时的偏导）

$$f(x,1) = x + (1-1)\sin\sqrt{\frac{x}{1}} = x + 0 = x$$

$$f_x(x,1) = \frac{d}{dx}(x) = 1$$

$$\boxed{答案：B（1）}$$

## 第2题：求偏导数

---

### 2(1) — 商的求导

$$z = \frac{x}{\sqrt{x^2+y^2}}$$

**求 $\frac{\partial z}{\partial x}$**（把 $y$ 当常数，用商的求导法则）：

$$\frac{\partial z}{\partial x} = \frac{1 \cdot \sqrt{x^2+y^2} - x \cdot \frac{x}{\sqrt{x^2+y^2}}}{x^2+y^2}$$

$$= \frac{x^2+y^2 - x^2}{(x^2+y^2)^{3/2}} = \boxed{\frac{y^2}{(x^2+y^2)^{3/2}}}$$

**求 $\frac{\partial z}{\partial y}$**（把 $x$ 当常数）：

$$\frac{\partial z}{\partial y} = x \cdot \left(-\frac{1}{2}\right)(x^2+y^2)^{-3/2} \cdot 2y$$

$$= \boxed{-\frac{xy}{(x^2+y^2)^{3/2}}}$$

---

### 2(2) — 复合函数求导

$$z = \ln\tan(xy)$$

**结构**：$\ln(\tan(\cdot))$，外层 $\ln$，内层 $\tan$，最内层 $xy$

**求 $\frac{\partial z}{\partial x}$**：

$$\frac{\partial z}{\partial x} = \frac{1}{\tan(xy)} \cdot \frac{1}{\cos^2(xy)} \cdot y$$

$$= \frac{y}{\sin(xy)\cos(xy)} = \boxed{\frac{2y}{\sin(2xy)}}$$

（用了 $\sin(2\theta) = 2\sin\theta\cos\theta$）

**求 $\frac{\partial z}{\partial y}$**：同理，把 $y$ 换成 $x$：

$$= \boxed{\frac{2x}{\sin(2xy)}}$$

---

### 2(3) — 指数函数求导

$$u = z^{x/y}$$

这是 $z$ 的幂次函数（$z$ 是底数，$x/y$ 是指数），把 $z$ 当变量，$x,y$ 当参数求偏导：

**求 $\frac{\partial u}{\partial z}$**：

$$\frac{\partial u}{\partial z} = \frac{x}{y} \cdot z^{x/y - 1} = \boxed{\frac{x}{y} z^{\frac{x}{y}-1}}$$

**求 $\frac{\partial u}{\partial x}$**（$z$ 当常数，$z^{x/y} = e^{\frac{x}{y}\ln z}$）：

$$\frac{\partial u}{\partial x} = z^{x/y} \cdot \frac{\ln z}{y} = \boxed{\frac{\ln z}{y} \cdot z^{x/y}}$$

**求 $\frac{\partial u}{\partial y}$**：

$$\frac{\partial u}{\partial y} = z^{x/y} \cdot \ln z \cdot \left(-\frac{x}{y^2}\right) = \boxed{-\frac{x\ln z}{y^2} \cdot z^{x/y}}$$

## 第3题 — 求特定点的偏导值

$$f(x,y) = e^{xy}\sin\pi y + (x-1)\arctan\sqrt{\frac{x}{y}}$$

求 $f_x(1,1)$ 和 $f_y(1,1)$

**技巧：先求偏导表达式，再代点；或者先代点简化再求导**

---

**求 $f_x$**（$y$ 当常数）：

$$\frac{\partial f}{\partial x} = ye^{xy}\sin\pi y + \arctan\sqrt{\frac{x}{y}} + (x-1) \cdot \frac{1}{1+\frac{x}{y}} \cdot \frac{1}{2\sqrt{\frac{x}{y}}} \cdot \frac{1}{y}$$

代入 $(1,1)$：

- $ye^{xy}\sin\pi y\big\rvert_{(1,1)} = 1 \cdot e^1 \cdot \sin\pi = 0$
- $\arctan\sqrt{x/y}\big\rvert_{(1,1)} = \arctan 1 = \dfrac{\pi}{4}$
- 第三项：$(x-1)\big|_{x=1} = 0$，整项为 $0$

$$\boxed{f_x(1,1) = \frac{\pi}{4}}$$

---

**求 $f_y$**（$x$ 当常数）：

$$\frac{\partial f}{\partial y} = xe^{xy}\sin\pi y + e^{xy}\cdot\pi\cos\pi y + (x-1)\cdot\frac{1}{1+x/y}\cdot\frac{1}{2\sqrt{x/y}}\cdot\left(-\frac{x}{y^2}\right)$$

代入 $(1,1)$：

- $xe^{xy}\sin\pi y\big\rvert_{(1,1)} = 1\cdot e\cdot\sin\pi = 0$
- $e^{xy}\cdot\pi\cos\pi y\big\rvert_{(1,1)} = e\cdot\pi\cdot\cos\pi = -\pi e$
- 第三项：$(x-1)\big|_{x=1} = 0$，整项为 $0$

$$\boxed{f_y(1,1) = -\pi e}$$

# 🗂️ 总结

| 情况 | 方法 |
|------|------|
| 一般点求偏导 | 把其他变量当常数，正常求导 |
| 特殊点（如原点）| 回到定义用极限 |
| $y=k$ 时的偏导 | 先代入 $y=k$，再对 $x$ 求导 |
| 求某点的值 | 先求偏导式，再代入坐标 |
| 复合函数 | 链式法则，从外到内逐层求导 |

## 🧠 二阶偏导数

**一阶偏导对一个变量求导，二阶就是再求一次**

四种二阶偏导：

| 符号 | 意思 |
|------|------|
| $\dfrac{\partial^2 z}{\partial x^2}$ 或 $z_{xx}$ | 对 $x$ 求两次 |
| $\dfrac{\partial^2 z}{\partial y^2}$ 或 $z_{yy}$ | 对 $y$ 求两次 |
| $\dfrac{\partial^2 z}{\partial x \partial y}$ 或 $z_{xy}$ | 先对 $x$，再对 $y$ |
| $\dfrac{\partial^2 z}{\partial y \partial x}$ 或 $z_{yx}$ | 先对 $y$，再对 $x$ |

> 🔑 **重要定理**：只要混合偏导连续，$z_{xy} = z_{yx}$（顺序可以交换！）

## 题4 — 曲线切线斜率

曲线：$\begin{cases} z = \sqrt{4-x^2-y^2} \\ x = 1 \end{cases}$，在点 $(1,-1,\sqrt{2})$ 处切线关于 $y$ 轴的斜率

**理解**：$x=1$ 固定，曲线在 $x=1$ 的截面上，切线斜率 = $\dfrac{\partial z}{\partial y}$

先求偏导：
$$\frac{\partial z}{\partial y} = \frac{1}{2\sqrt{4-x^2-y^2}} \cdot (-2y) = \frac{-y}{\sqrt{4-x^2-y^2}}$$

代入点 $(1,-1,\sqrt{2})$：
$$\frac{\partial z}{\partial y}\bigg|_{(1,-1)} = \frac{-(-1)}{\sqrt{4-1-1}} = \frac{1}{\sqrt{2}}$$

$$\boxed{\text{斜率} = \frac{\sqrt{2}}{2}}$$

## 题5(1) — 求全部二阶偏导

$$z = \sin^2(ax+by)$$

**第一步：求一阶偏导**

令 $u = ax+by$，则 $z = \sin^2 u$

$$z_x = 2\sin u \cdot \cos u \cdot a = a\sin(2u) = a\sin 2(ax+by)$$

$$z_y = b\sin 2(ax+by)$$

**第二步：求二阶偏导**

$$z_{xx} = a \cdot \cos 2(ax+by) \cdot 2a = \boxed{2a^2\cos 2(ax+by)}$$

$$z_{yy} = \boxed{2b^2\cos 2(ax+by)}$$

$$z_{xy} = a \cdot \cos 2(ax+by) \cdot 2b = \boxed{2ab\cos 2(ax+by)}$$

$$z_{yx} = z_{xy} = \boxed{2ab\cos 2(ax+by)}$$

## 题5(2) — 混合偏导

$$z = e^{x^2/y},\quad \text{求} \frac{\partial^2 z}{\partial x \partial y}$$

**先对 $x$ 求偏导**（$y$ 当常数）：

$$\frac{\partial z}{\partial x} = e^{x^2/y} \cdot \frac{2x}{y}$$

**再对 $y$ 求偏导**（用乘积法则）：

$$\frac{\partial^2 z}{\partial y \partial x} = \frac{2x}{y} \cdot e^{x^2/y} \cdot \left(-\frac{x^2}{y^2}\right) + e^{x^2/y} \cdot \left(-\frac{2x}{y^2}\right)$$

$$= e^{x^2/y} \cdot \frac{2x}{y^2}\left(-\frac{x^2}{y} - 1\right)$$

$$= \boxed{-\frac{2x(x^2+y)}{y^3} e^{x^2/y}}$$

## 题6 — 三阶混合偏导

$$f(x,y,z) = \ln(x^2+y^2+z^2),\quad \text{求} f_{xyz}(1,1,1)$$

**逐步求导**，令 $r^2 = x^2+y^2+z^2$

**第一步：对 $x$ 求导**

$$f_x = \frac{2x}{x^2+y^2+z^2}$$

**第二步：对 $y$ 求导**

$$f_{xy} = 2x \cdot \frac{-2y}{(x^2+y^2+z^2)^2} = \frac{-4xy}{(x^2+y^2+z^2)^2}$$

**第三步：对 $z$ 求导**（用商的求导）

$$f_{xyz} = -4xy \cdot \frac{-2z}{(x^2+y^2+z^2)^2} \cdot \frac{(x^2+y^2+z^2)^2 - 0}{(...)^2}$$

$$= \frac{-4xy \cdot (-2z)}{(x^2+y^2+z^2)^3} \cdot (x^2+y^2+z^2)$$

等等，更直接：

$$f_{xyz} = \frac{\partial}{\partial z}\left[\frac{-4xy}{(x^2+y^2+z^2)^2}\right] = -4xy \cdot \frac{-2 \cdot 2z}{(x^2+y^2+z^2)^3} \cdot (x^2+y^2+z^2)$$

$$= \frac{16xyz}{(x^2+y^2+z^2)^3}$$

代入 $(1,1,1)$：

$$f_{xyz}(1,1,1) = \frac{16\cdot1\cdot1\cdot1}{(1+1+1)^3} = \frac{16}{27}$$

$$\boxed{f_{xyz}(1,1,1) = \frac{16}{27}}$$

# 习题 9-3：全微分

## 全微分是什么？

**一元函数**：$dy = f'(x)dx$（用切线近似曲线）

**二元函数**：$dz = \dfrac{\partial z}{\partial x}dx + \dfrac{\partial z}{\partial y}dy$

> 直觉：$z$ 的微小变化 = $x$ 方向的贡献 + $y$ 方向的贡献

**全增量 vs 全微分**：
- 全增量：$\Delta z = f(x_0+\Delta x, y_0+\Delta y) - f(x_0, y_0)$（精确值）
- 全微分：$dz = z_x dx + z_y dy$（近似值，$dx=\Delta x, dy=\Delta y$）

## 🔑 连续、偏导、可微的关系

```
可微 ──→ 连续
 │
 └──→ 偏导数存在

偏导数存在 ≠ 连续，≠ 可微
连续 ≠ 偏导数存在
偏导数连续 ──→ 可微（充分条件）
```

### 1(1) 连续 → ?

连续 **不能推出** 可偏导，也不能推出可微

### 1(2) 偏导数存在 → ?

偏导存在 **不能推出** 连续，更不能推出可微

## 题2 — 计算全增量和全微分

$z = xy$，$\Delta x=0.2$，$\Delta y=-0.1$，$x_0=1$，$y_0=2$

**全增量**（精确）：

$$\Delta z = (x_0+\Delta x)(y_0+\Delta y) - x_0 y_0$$
$$= 1.2 \times 1.9 - 1 \times 2 = 2.28 - 2 = \boxed{0.28}$$

**全微分**（近似）：

$$\frac{\partial z}{\partial x} = y,\quad \frac{\partial z}{\partial y} = x$$

$$dz = y_0\,\Delta x + x_0\,\Delta y = 2\times0.2 + 1\times(-0.1) = 0.4 - 0.1 = \boxed{0.3}$$

## 题3 — 求全微分

### 3(1) $z = \arctan\dfrac{y}{x}$

$$\frac{\partial z}{\partial x} = \frac{1}{1+(y/x)^2} \cdot \left(-\frac{y}{x^2}\right) = \frac{x^2}{x^2+y^2} \cdot \left(-\frac{y}{x^2}\right) = -\frac{y}{x^2+y^2}$$

$$\frac{\partial z}{\partial y} = \frac{1}{1+(y/x)^2} \cdot \frac{1}{x} = \frac{x^2}{x^2+y^2} \cdot \frac{1}{x} = \frac{x}{x^2+y^2}$$

$$\boxed{dz = -\frac{y}{x^2+y^2}\,dx + \frac{x}{x^2+y^2}\,dy}$$

### 3(2) $z = \dfrac{xy^2}{\sqrt{x^2+y^2}}$

**求 $z_x$**（商的求导，$y$ 当常数）：

$$z_x = \frac{y^2\sqrt{x^2+y^2} - xy^2 \cdot \frac{x}{\sqrt{x^2+y^2}}}{x^2+y^2}$$

$$= \frac{y^2(x^2+y^2) - x^2y^2}{(x^2+y^2)^{3/2}} = \frac{y^4}{(x^2+y^2)^{3/2}}$$

**求 $z_y$**（$x$ 当常数）：

$$z_y = \frac{2xy\sqrt{x^2+y^2} - xy^2 \cdot \frac{y}{\sqrt{x^2+y^2}}}{x^2+y^2}$$

$$= \frac{2xy(x^2+y^2) - xy^3}{(x^2+y^2)^{3/2}} = \frac{xy(2x^2+y^2)}{(x^2+y^2)^{3/2}}$$

$$\boxed{dz = \frac{y^4}{(x^2+y^2)^{3/2}}dx + \frac{xy(2x^2+y^2)}{(x^2+y^2)^{3/2}}dy}$$

# 9-4 多元复合函数求导
多元复合函数求导的核心在于“路径搜索”。只要弄清楚自变量如何通过中间变量影响最终函数。
### 一、 核心法则：链式法则
 1. **分叉相加，路径相乘**：
   * 从因变量（如 z）到自变量（如 t）有几条路，就分几项相加。
   * 在每一条路径上，每经过一个层级，就求一次偏导并相乘。
 2. **中间变量的识别**：
   * 如果 z = f(u, v)，而 u, v 又是 x, y 的函数，则 u, v 是**中间变量**，x, y 是**自变量**。
   * 此时，\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial x}。
### 二、 117页详解
#### 1. 显式一元复合
**题目**：z=2^{xe^y}, x=t^3, y=\sin t, 求 \frac{dz}{dt}。
 * **路径**：z \to x \to t 以及 z \to y \to t。
 * **步骤**：
   1.    2.    3.  * **结果**：\frac{dz}{dt} = (2^{xe^y} \ln 2 \cdot e^y)(3t^2) + (2^{xe^y} \ln 2 \cdot xe^y)(\cos t)。
#### 2. 显式多元复合
**题目**：z=\sin(2u+3v), u=xy, v=x^2+y^2, 求 \frac{\partial z}{\partial x}, \frac{\partial z}{\partial y}。
 * **求 \frac{\partial z}{\partial x}**：
   \frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} u_x + \frac{\partial z}{\partial v} v_x = \cos(2u+3v) \cdot 2 \cdot y + \cos(2u+3v) \cdot 3 \cdot 2x。
 * **求 \frac{\partial z}{\partial y}**：
   \frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} u_y + \frac{\partial z}{\partial v} v_y = \cos(2u+3v) \cdot 2 \cdot x + \cos(2u+3v) \cdot 3 \cdot 2y。
#### 3. 幂指函数偏导
**题目**：z=(2x+y)^{-x}。
 * **方法**：化为 z = e^{-x \ln(2x+y)}。
 * **求 \frac{\partial z}{\partial y}**（较简单）：
   \frac{\partial z}{\partial y} = e^{-x \ln(2x+y)} \cdot [ -x \cdot \frac{1}{2x+y} \cdot 1 ] = -\frac{x(2x+y)^{-x}}{2x+y}。
#### 4. 抽象函数一阶导
**题目**：(1) z=f(xy^2, \frac{x^2}{y})。
 * **法则**：设 u=xy^2, v=\frac{x^2}{y}。
 * **结果**：\frac{\partial z}{\partial x} = f'_1 \cdot y^2 + f'_2 \cdot \frac{2x}{y}；\frac{\partial z}{\partial y} = f'_1 \cdot 2xy + f'_2 \cdot (-\frac{x^2}{y^2})。
#### 5. 全微分
**题目**：u=f(xy, \frac{x}{z}), 求 du。
 * **结果**：先求 u_x, u_y, u_z。
   u_x = f'_1 y + f'_2 \frac{1}{z}, \quad u_y = f'_1 x, \quad u_z = f'_2 (-\frac{x}{z^2})。
   du = (f'_1 y + \frac{f'_2}{z})dx + (f'_1 x)dy - \frac{xf'_2}{z^2}dz。
### 三、 118-119页：高阶与综合题目详解
#### 7. 二阶导数（以第(2)小题为例）
**题目**：z=f(x^2+y^2), 求 \frac{\partial^2 z}{\partial x^2}。
 1. **一阶**：\frac{\partial z}{\partial x} = f' \cdot 2x。
 2. **二阶**（乘法法则）：\frac{\partial^2 z}{\partial x^2} = \frac{\partial}{\partial x}(f') \cdot 2x + f' \cdot \frac{\partial}{\partial x}(2x)。
 3. **内层再复合**：\frac{\partial}{\partial x}(f') = f'' \cdot 2x。
 4. **最终**：4x^2 f'' + 2f'。
#### 8. 乘积形式的混合导数
**题目**：u=\frac{1}{x}f(x, xy), 求 \frac{\partial^2 u}{\partial x \partial y}。
 1. **先对 y 求导**：\frac{\partial u}{\partial y} = \frac{1}{x} \cdot f'_2 \cdot x = f'_2(x, xy)。
 2. **再对 x 求导**：\frac{\partial}{\partial x}[f'_2(x, xy)] = f''_{21} \cdot 1 + f''_{22} \cdot y。
#### 10. 多层嵌套
**题目**：z=f[\phi(x-y), \sin y], 求 \frac{\partial^2 z}{\partial x \partial y}。
 1. **一阶 \frac{\partial z}{\partial x}**：f'_1 \cdot \phi' \cdot 1。
 2. **二阶 \frac{\partial}{\partial y}(f'_1 \cdot \phi')**：
   * 注意 f'_1 的两个位置都含 y，\phi' 也含 y。
   * 结果：[f''_{11} \cdot \phi' \cdot (-1) + f''_{12} \cdot \cos y] \cdot \phi' + f'_1 \cdot \phi'' \cdot (-1)。

| 题目类型 | 关键技巧 | 易错点 |
|---|---|---|
| **具体解析式** | 直接套链式公式 | 忘记对括号内的项求导（内导数） |
| **抽象函数 f** | 使用 f'_1, f'_2 代号 | 二阶导时忘记 f'_1 仍然是复合函数 |
| **高阶导数** | 结合乘法法则 (uv)' | 漏掉“前导后不导”中的某一项 |

# 9-5 隐函数求导法则
隐与显函数的区别在于：变量之间搅和在一起，没有孤立出因变量。求导的核心逻辑是**“整体微分”**或**“偏导比例”**。
### 一、 什么是隐函数求导？
 1. **存在性逻辑**：方程 F(x, y) = 0 在满足一定条件下，虽然没写成 y=...，但它在局部轨迹上确定了一个函数。
 2. **公式法的本质（负号由来）**：
   对 F(x, y) = 0 两边求全微分：dF = F'_x dx + F'_y dy = 0。
   移项得：F'_y dy = -F'_x dx \implies \frac{dy}{dx} = -\frac{F'_x}{F'_y}。
   这就是为什么公式里必然带一个**负号**，且下标**上下颠倒**。

### 二、 120页详解
#### 1. 基础方程求偏导
**题目** (1)：设 \frac{x}{y} + \ln(zy) = 0，求 \frac{\partial z}{\partial x}, \frac{\partial z}{\partial y}。
 * **方法**：构造 F(x, y, z) = x y^{-1} + \ln z + \ln y = 0。
 * **步骤**：
   1.    2.    3.  * **结果**：
   \frac{\partial z}{\partial x} = -\frac{F'_x}{F'_z} = -\frac{1/y}{1/z} = -\frac{z}{y}。
   \frac{\partial z}{\partial y} = -\frac{F'_y}{F'_z} = -\frac{(y-x)/y^2}{1/z} = \frac{z(x-y)}{y^2}。
**题目** (2)：设 z = x + y\phi(z)，求 dz。
 * **方法**：直接两边微分（全微分法）。
 * **步骤**：
   1.    2. 将含有 dz 的项移到左边：dz(1 - y\phi'(z)) = dx + \phi(z)dy
 * **结果**：dz = \frac{1}{1-y\phi'(z)}dx + \frac{\phi(z)}{1-y\phi'(z)}dy。
#### 2. 二阶隐偏导
**题目** (2)：设 e^z - xyz = 0，求 \frac{\partial^2 z}{\partial x \partial y}。
 * **一阶**：对 x 求偏导（视 z 为 z(x,y)）：
   e^z z_x - (yz + xyz_x) = 0 \implies z_x(e^z - xy) = yz \implies z_x = \frac{yz}{e^z - xy}。
 * **二阶**：对上式再关于 y 求导（注意分子分母的 z 都是 y 的函数）：
   使用分式法则：\frac{\partial}{\partial y}(\frac{yz}{e^z - xy}) = \frac{(z + yz_y)(e^z - xy) - yz(e^z z_y - x)}{(e^z - xy)^2}。
 * **收尾**：将 z_y = \frac{xz}{e^z - xy} 代入上式整理即可。
### 三、 121页：隐函数方程组与证明题
#### 3. 方程组求导
**题目** (1)：\begin{cases} z = x^2 + y^2 \\ x^2 + 2y^2 + z^2 = 1 \end{cases}，求 \frac{dz}{dx}, \frac{dy}{dx}。
 * **方法**：两边同时对自变量 x 求导，建立联立方程。
 * **步骤**：
   1.    2.  * **解方程**：将 (1) 式的 2yy' = z' - 2x 代入 (2) 式：
   2x + 2(z' - 2x) + 2zz' = 0 \implies 2x + 2z' - 4x + 2zz' = 0 \implies 2z'(1+z) = 2x。
 * **结果**：\frac{dz}{dx} = \frac{x}{1+z}。求出 z' 后代回 (1) 即可求得 y'。
#### 4. 抽象隐函数证明
**题目**：F(\frac{y}{x}, \frac{z}{x}) = 0，证明 x z_x + y z_y = z。
 * **概念引入：齐次函数背景**。此方程隐含 z 是关于 x, y 的一阶齐次函数。
 * **证明逻辑**：
   1. 设 u = \frac{y}{x}, v = \frac{z}{x}。则 F(u, v) = 0。
   2. z_x = -\frac{F'_u u_x + F'_v v_x}{F'_v v_z} = -\frac{F'_u(-\frac{y}{x^2}) + F'_v(\frac{z_x x - z}{x^2})}{F'_v(\frac{1}{x})}。
   3. 化简后你会发现，将 z_x 和 z_y 代入左式，所有 F'_u, F'_v 项会抵消。
